# 分割回文串（Palindrome Partitioning）

- 题目链接：https://leetcode.cn/problems/palindrome-partitioning/
- 难度：中等
- 标签：字符串、回溯、动态规划

## 题目描述

给你一个字符串 `s`，请你将 `s` 分割成一些子串，使每个子串都是**回文串**。返回 `s` 所有可能的分割方案。

### 示例

```
输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]

输入：s = "a"
输出：[["a"]]

输入：s = "aaa"
输出：[["a","a","a"],["a","aa"],["aa","a"],["aaa"]]
```

### 约束

- `1 <= s.length <= 16`
- `s` 仅由小写英文字母组成

---

## 解法一：回溯 + DP 预处理回文（推荐）

### 思路

先用 DP 预处理 `is_pal[i][j]` 表示 `s[i:j+1]` 是否回文。回溯时从 `start` 枚举切分点 `end`，若 `s[start:end+1]` 是回文则加入 path，递归 `start = end+1`。

### 代码

```python
from typing import List


class Solution:
    def partition(self, s: str) -> List[List[str]]:
        n = len(s)
        is_pal = [[False] * n for _ in range(n)]
        for i in range(n - 1, -1, -1):
            for j in range(i, n):
                if s[i] == s[j] and (j - i <= 2 or is_pal[i + 1][j - 1]):
                    is_pal[i][j] = True

        res, path = [], []

        def backtrack(start: int) -> None:
            if start == n:
                res.append(path[:])
                return
            for end in range(start, n):
                if not is_pal[start][end]:
                    continue
                path.append(s[start : end + 1])
                backtrack(end + 1)
                path.pop()

        backtrack(0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × 2^n + n²)，预处理 + 回溯 |
| 空间 | O(n²)，回文表 + O(n) 递归 |

### 讲解

回文判断若每次 O(n)，总复杂度会爆炸。O(n²) 预处理是标准优化。回溯本质是「切分点选择」。

## 解法二：回溯 + 双指针即时判断

### 思路

不预处理，每次切分时用双指针 O(n) 判断是否回文。代码简单，n ≤ 16 可接受。

### 代码

```python
from typing import List


class Solution:
    def partition(self, s: str) -> List[List[str]]:
        n = len(s)
        res, path = [], []

        def is_pal(l: int, r: int) -> bool:
            while l < r:
                if s[l] != s[r]:
                    return False
                l += 1
                r -= 1
            return True

        def backtrack(start: int) -> None:
            if start == n:
                res.append(path[:])
                return
            for end in range(start, n):
                if not is_pal(start, end):
                    continue
                path.append(s[start : end + 1])
                backtrack(end + 1)
                path.pop()

        backtrack(0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × 2^n × n) |
| 空间 | O(n) |

### 讲解

适合快速实现。n 较大时应换预处理表。

## 解法三：记忆化搜索（自顶向下 DP）

### 思路

定义 `dfs(i)` 为 `s[i:]` 的所有分割方案。枚举第一个回文前缀 `s[i:j+1]`，拼接 `dfs(j+1)` 的结果。

### 代码

```python
from typing import List


class Solution:
    def partition(self, s: str) -> List[List[str]]:
        n = len(s)
        is_pal = [[False] * n for _ in range(n)]
        for i in range(n - 1, -1, -1):
            for j in range(i, n):
                if s[i] == s[j] and (j - i <= 2 or is_pal[i + 1][j - 1]):
                    is_pal[i][j] = True

        memo = {}

        def dfs(i: int) -> List[List[str]]:
            if i == n:
                return [[]]
            if i in memo:
                return memo[i]
            res = []
            for j in range(i, n):
                if is_pal[i][j]:
                    for rest in dfs(j + 1):
                        res.append([s[i : j + 1]] + rest)
            memo[i] = res
            return res

        return dfs(0)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × 2^n + n²) |
| 空间 | O(n × 2^n) |

### 讲解

与回溯等价的不同视角。记忆化避免重复计算同一后缀，但存储所有方案时空间仍大。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 预处理+回溯 | O(n×2^n) | O(n²) | 面试首选 |
| 即时判断 | O(n²×2^n) | O(n) | n 小、代码短 |
| 记忆化 DFS | O(n×2^n) | O(输出) | 理解 DP 视角 |

---

## 常见错误

1. **回文 DP 方向错误**：应从后往前填 `is_pal[i][j]`，依赖 `is_pal[i+1][j-1]`。
2. **切分后 start 未更新**：应递归 `backtrack(end + 1)`。
3. **空串回文**：长度 0 子串在本题切分中不会出现（始终取非空前缀）。
4. **与「最长回文子串」混淆**：5 题求最长子串，131 题求所有切分方案。
