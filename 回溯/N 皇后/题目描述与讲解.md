# N 皇后（N-Queens）

- 题目链接：https://leetcode.cn/problems/n-queens/
- 难度：困难
- 标签：数组、回溯

## 题目描述

按照国际象棋的规则，皇后可以攻击与之处在同一行、同一列、同一斜线上的棋子。

`n` 皇后问题研究的是如何将 `n` 个皇后放置在 `n × n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n`，返回所有不同的 **n 皇后问题** 的解决方案。每种解法包含一个不同的 **n 皇后问题** 的棋子放置方案，该方案中 `'Q'` 和 `'.'` 分别代表皇后和空位。

### 示例

```
输入：n = 4
输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]

输入：n = 1
输出：[["Q"]]
```

### 约束

- `1 <= n <= 9`

---

## 解法一：回溯 + 集合记录冲突（推荐）

### 思路

逐行放皇后，每行只放一个。用三个集合记录已占用的列、主对角线 `row-col`、副对角线 `row+col`。到第 `n` 行时收集棋盘。

### 代码

```python
from typing import List


class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res, board = [], ["." * n for _ in range(n)]
        cols, diag1, diag2 = set(), set(), set()

        def backtrack(row: int) -> None:
            if row == n:
                res.append(board[:])
                return
            for col in range(n):
                d1, d2 = row - col, row + col
                if col in cols or d1 in diag1 or d2 in diag2:
                    continue
                cols.add(col)
                diag1.add(d1)
                diag2.add(d2)
                board[row] = "." * col + "Q" + "." * (n - col - 1)
                backtrack(row + 1)
                cols.remove(col)
                diag1.remove(d1)
                diag2.remove(d2)

        backtrack(0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n!)，逐行尝试列，冲突剪枝 |
| 空间 | O(n)，集合 + 递归栈 |

### 讲解

关键洞察：每行恰好一个皇后，只需枚举列。对角线用 `row±col` 标识，O(1) 判冲突。

## 解法二：回溯 + 位运算（n ≤ 32 时高效）

### 思路

用三个位掩码 `cols`、`diag1`、`diag2` 表示列和两条对角线占用。当前行可选列 = `~(cols | diag1 | diag2) & ((1<<n)-1)`。

### 代码

```python
from typing import List


class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res = []
        board = [0] * n  # board[row] = col index

        def backtrack(row: int, cols: int, d1: int, d2: int) -> None:
            if row == n:
                res.append(["." * c + "Q" + "." * (n - c - 1) for c in board])
                return
            avail = ~(cols | d1 | d2) & ((1 << n) - 1)
            while avail:
                bit = avail & -avail
                col = bit.bit_length() - 1
                board[row] = col
                backtrack(row + 1, cols | bit, (d1 | bit) << 1, (d2 | bit) >> 1)
                avail -= bit

        backtrack(0, 0, 0, 0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n!) |
| 空间 | O(n) |

### 讲解

位运算常数更小，是 N 皇后经典优化。理解 `d1<<1`、`d2>>1` 表示对角线逐行延伸。

## 解法三：暴力回溯 + 棋盘检查

### 思路

每格尝试放 Q，放满 n 个后检查棋盘是否合法。效率低，仅作理解。

### 代码

```python
from typing import List


class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        res = []
        board = [["."] * n for _ in range(n)]

        def valid(r, c):
            for i in range(n):
                if board[i][c] == "Q" or board[r][i] == "Q":
                    return False
            for i in range(n):
                if 0 <= r - i < n and 0 <= c - i < n and board[r - i][c - i] == "Q":
                    return False
                if 0 <= r + i < n and 0 <= c + i < n and board[r + i][c + i] == "Q":
                    return False
            return True

        def backtrack(cnt):
            if cnt == n:
                res.append(["".join(row) for row in board])
                return
            for r in range(n):
                for c in range(n):
                    if board[r][c] == "." and valid(r, c):
                        board[r][c] = "Q"
                        backtrack(cnt + 1)
                        board[r][c] = "."

        backtrack(0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n^(n²)) 量级，极慢 |
| 空间 | O(n²) |

### 讲解

帮助理解约束，但 n=4 以上就会超时。必须用逐行+冲突集合剪枝。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 集合回溯 | O(n!) | O(n) | 面试首选 |
| 位运算 | O(n!) | O(n) | 追求常数优化 |
| 暴力棋盘 | 极慢 | O(n²) | 理解约束 |

## 常见错误

1. **对角线标识错误**：主对角线 `row-col`，副对角线 `row+col`。
2. **同行多皇后**：应逐行递归，而非逐格随意放。
3. **忘记撤销集合**：回溯必须 remove。
4. **与「N 皇后 II」混淆**：52 题只求方案数，不需输出棋盘。
