# 电话号码的字母组合（Letter Combinations of a Phone Number）

- 题目链接：https://leetcode.cn/problems/letter-combinations-of-a-phone-number/
- 难度：中等
- 标签：哈希表、字符串、回溯

## 题目描述

给定一个仅包含数字 `2-9` 的字符串，返回所有它能表示的字母组合。数字到字母的映射与电话按键相同（不含 0 和 1）。

```
2 → abc   3 → def   4 → ghi   5 → jkl
6 → mno   7 → pqrs  8 → tuv   9 → wxyz
```

注意：`1` 不对应任何字母。

### 示例

```
输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]

输入：digits = ""
输出：[]

输入：digits = "2"
输出：["a","b","c"]
```

### 约束

- `0 <= digits.length <= 4`
- `digits[i]` 是 `2-9` 中的一个数字

---

## 解法一：回溯（推荐）

### 思路

按数字下标 `i` 逐层展开。到达 `i == len(digits)` 时，将 `path` 拼成字符串加入结果。每层遍历当前数字对应的所有字母，选择 → 递归 → 撤销。

### 代码

```python
from typing import List


class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        mapping = {
            "2": "abc", "3": "def", "4": "ghi", "5": "jkl",
            "6": "mno", "7": "pqrs", "8": "tuv", "9": "wxyz",
        }
        res, path = [], []

        def backtrack(i: int) -> None:
            if i == len(digits):
                res.append("".join(path))
                return
            for ch in mapping[digits[i]]:
                path.append(ch)
                backtrack(i + 1)
                path.pop()

        backtrack(0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(3^n × 4^m)，n 为对应 3 字母的数字个数，m 为 4 字母 |
| 空间 | O(n)，递归栈 + path |

### 讲解

本质是**多叉树 DFS**。`digits` 为空时直接返回 `[]`，这是边界条件。回溯模板与全排列类似，但这里是「按层选字母」而非「选未用元素」。

## 解法二：迭代 BFS/队列

### 思路

用队列维护当前已生成的组合前缀。每处理一个数字，把队列中每个前缀分别接上该数字的所有字母，形成新前缀。

### 代码

```python
from collections import deque
from typing import List


class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        mapping = {
            "2": "abc", "3": "def", "4": "ghi", "5": "jkl",
            "6": "mno", "7": "pqrs", "8": "tuv", "9": "wxyz",
        }
        q = deque([""])
        for d in digits:
            for _ in range(len(q)):
                prefix = q.popleft()
                for ch in mapping[d]:
                    q.append(prefix + ch)
        return list(q)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(3^n × 4^m) |
| 空间 | O(3^n × 4^m)，存储所有中间前缀 |

### 讲解

层序遍历视角：每处理一位数字，组合数乘以对应字母数。思路直观，但空间占用比回溯大。

## 解法三：递归直接拼接（无 path 数组）

### 思路

递归返回「从 index 开始的所有后缀组合」，当前层把本层每个字母与后缀拼接。

### 代码

```python
from typing import List


class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        if not digits:
            return []
        mapping = {
            "2": "abc", "3": "def", "4": "ghi", "5": "jkl",
            "6": "mno", "7": "pqrs", "8": "tuv", "9": "wxyz",
        }

        def dfs(i: int) -> List[str]:
            if i == len(digits):
                return [""]
            res = []
            for ch in mapping[digits[i]]:
                for suffix in dfs(i + 1):
                    res.append(ch + suffix)
            return res

        return dfs(0)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(3^n × 4^m) |
| 空间 | O(n)，递归栈 |

### 讲解

函数式写法，不显式维护 path 和回溯。适合理解「当前选择 × 后续所有组合」的笛卡尔积结构。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 回溯 | O(3^n × 4^m) | O(n) | 面试首选 |
| BFS 队列 | O(3^n × 4^m) | O(输出) | 层序思维 |
| 递归拼接 | O(3^n × 4^m) | O(n) | 函数式、无显式回溯 |

---

## 常见错误

1. **空字符串未处理**：`digits == ""` 应返回 `[]`，不是 `[""]`。
2. **映射表遗漏 "7"/"9"**：它们有 4 个字母，其余 3 个。
3. **忘记回溯 pop**：path 会累积错误字符。
4. **与「括号生成」混淆**：本题是固定映射的多叉树，无需剪枝。
