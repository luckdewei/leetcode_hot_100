# 单词搜索（Word Search）

- 题目链接：https://leetcode.cn/problems/word-search/
- 难度：中等
- 标签：数组、字符串、回溯、矩阵

## 题目描述

给定一个 `m × n` 二维字符网格 `board` 和一个字符串 `word`，如果 `word` 存在于网格中，返回 `true`；否则返回 `false`。

单词必须按照字母顺序，通过**相邻**的单元格内的字母构成，其中「相邻」单元格是那些水平相邻或垂直相邻的单元格。**同一个单元格内的字母不允许被重复使用**。

### 示例

```
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
输出：true

输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "SEE"
输出：true

输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCB"
输出：false
```

### 约束

- `m == board.length`，`n == board[i].length`
- `1 <= m, n <= 6`
- `1 <= word.length <= 15`
- `board` 和 `word` 仅由大小写英文字母组成

---

## 解法一：DFS 回溯 + 原地标记（推荐）

### 思路

从每个格子出发 DFS。匹配 `word[k]` 时，临时将 `board[r][c]` 改为 `"#"` 防止重复使用，四方向递归，结束后恢复。

### 代码

```python
from typing import List


class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        m, n = len(board), len(board[0])

        def dfs(r: int, c: int, k: int) -> bool:
            if k == len(word):
                return True
            if r < 0 or r >= m or c < 0 or c >= n or board[r][c] != word[k]:
                return False
            tmp, board[r][c] = board[r][c], "#"
            found = (
                dfs(r + 1, c, k + 1)
                or dfs(r - 1, c, k + 1)
                or dfs(r, c + 1, k + 1)
                or dfs(r, c - 1, k + 1)
            )
            board[r][c] = tmp
            return found

        for i in range(m):
            for j in range(n):
                if dfs(i, j, 0):
                    return True
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn × 4^L)，L 为 word 长度 |
| 空间 | O(L)，递归栈 |

### 讲解

网格 DFS 回溯经典题。用 `"#"` 标记比 `visited` 数组更省空间，但会修改输入（已恢复）。

## 解法二：DFS + visited 数组

### 思路

与解法一相同，但用 `visited[r][c]` 记录是否访问，不修改 board。

### 代码

```python
from typing import List


class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        m, n = len(board), len(board[0])
        visited = [[False] * n for _ in range(m)]

        def dfs(r: int, c: int, k: int) -> bool:
            if k == len(word):
                return True
            if r < 0 or r >= m or c < 0 or c >= n:
                return False
            if visited[r][c] or board[r][c] != word[k]:
                return False
            visited[r][c] = True
            ok = any(dfs(r + dr, c + dc, k + 1) for dr, dc in ((1,0),(-1,0),(0,1),(0,-1)))
            visited[r][c] = False
            return ok

        for i in range(m):
            for j in range(n):
                if dfs(i, j, 0):
                    return True
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn × 4^L) |
| 空间 | O(mn + L) |

### 讲解

不修改原数组，语义更清晰。多 O(mn) 空间，面试两种写法均可。

## 解法三：Trie + 网格 DFS（扩展）

### 思路

单 word 搜索用 DFS 足够。若需同时搜多个 word，先把 words 建 Trie，DFS 时沿 Trie 走，不匹配则剪枝。以下为单 word 的 Trie 框架示意。

### 代码

```python
from typing import List


class TrieNode:
    def __init__(self):
        self.children = {}
        self.word = None


class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        root = TrieNode()
        node = root
        for ch in word:
            node = node.children.setdefault(ch, TrieNode())
        node.word = word

        m, n = len(board), len(board[0])

        def dfs(r, c, node):
            if node.word:
                return True
            ch = board[r][c]
            if ch not in node.children:
                return False
            board[r][c] = "#"
            nxt = node.children[ch]
            for dr, dc in ((1,0),(-1,0),(0,1),(0,-1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < m and 0 <= nc < n and dfs(nr, nc, nxt):
                    board[r][c] = ch
                    return True
            board[r][c] = ch
            return False

        for i in range(m):
            for j in range(n):
                if dfs(i, j, root):
                    return True
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn × 4^L) |
| 空间 | O(L) Trie + O(L) 栈 |

### 讲解

单 word 时 Trie 无优势；**单词搜索 II**（212）多词共用前缀时才必须用 Trie。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 原地标记 | O(mn×4^L) | O(L) | 面试首选 |
| visited 数组 | O(mn×4^L) | O(mn+L) | 不修改输入 |
| Trie DFS | O(mn×4^L) | O(L) | 多词搜索 II |

## 常见错误

1. **忘记恢复 board/visited**：导致后续起点搜索失败。
2. **对角线移动**：只能上下左右，不能斜走。
3. **起点未枚举**：必须从每个 `(i,j)` 尝试，不能只从 `(0,0)` 开始。
4. **提前返回未回溯**：应用 `or` 链或显式恢复后再 return。
