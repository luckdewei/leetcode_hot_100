# 不同路径（Unique Paths）

- 题目链接：https://leetcode.cn/problems/unique-paths/
- 难度：中等
- 标签：数学、动态规划、组合数学

## 题目描述

一个机器人位于 `m × n` 网格的左上角（起始点标记为 `Start`）。机器人每次只能向下或向右移动一步，试图到达网格的右下角。

问：从左上角到右下角总共有多少条不同的路径？

### 示例

```
输入：m = 3, n = 7
输出：28

输入：m = 3, n = 2
输出：3
解释：三条路径为 右→下→下、下→右→下、下→下→右
```

### 约束

- `1 <= m, n <= 100`

---

## 解法一：二维 DP

### 思路

1. `dp[i][j]`：从 (0,0) 到 (i,j) 的路径数。
2. 第一行、第一列均为 1（只能一直右或一直下）。
3. `dp[i][j] = dp[i-1][j] + dp[i][j-1]`。

### 代码

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1] * n for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[m - 1][n - 1]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m × n) |
| 空间 | O(m × n) |

### 讲解

经典网格 DP 入门题。每个格子只依赖上方和左方，适合理解二维状态压缩。

## 解法二：一维滚动（推荐，与 solution.py 一致）

### 思路

1. 用长度 n 的数组 `dp`，`dp[j]` 表示当前行到第 j 列的路径数。
2. 第一行全 1；之后每行：`dp[j] += dp[j-1]`（当前来自上方 + 左方）。
3. `m==1` 时直接返回 1。

### 代码

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [1] * n
        for _ in range(1, m):
            for j in range(1, n):
                dp[j] += dp[j - 1]
        return dp[0] if m == 1 else dp[-1]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m × n) |
| 空间 | O(n) |

### 讲解

与 `solution.py` 一致。内层 j 正序即可，因为 `dp[j-1]` 已是本行左方更新后的值。

## 解法三：组合数学

### 思路

共需走 `m-1` 步下、`n-1` 步右，共 `m+n-2` 步，选 `m-1` 步为下（或 `n-1` 步为右），答案 C(m+n-2, m-1)。

### 代码

```python
import math


class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        return math.comb(m + n - 2, m - 1)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(min(m,n)) 求组合数 |
| 空间 | O(1) |

### 讲解

最优时间，但「不同路径 II」有障碍时不能直接用。面试先写 DP 再提组合数加分。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 二维 DP | O(mn) | O(mn) | 理解转移 |
| 一维滚动 | O(mn) | O(n) | 面试首选 |
| 组合数 | O(min(m,n)) | O(1) | 无障碍网格 |

## 常见错误

1. **初始化**：第一行第一列是 1，不是 0。
2. **m=1 或 n=1**：只有一条路径，滚动数组写法需特判或自然得到 1。
3. **与 63 题混淆**：63 题有障碍物，需 DP 而非纯组合。
