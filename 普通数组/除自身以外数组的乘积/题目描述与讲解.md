# 除自身以外数组的乘积（Product of Array Except Self）

- 题目链接：https://leetcode.cn/problems/product-of-array-except-self/
- 难度：中等
- 标签：数组、前缀和

## 题目描述

给你一个整数数组 `nums`，返回数组 `answer`，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积。

题目数据保证数组 `[1,n]` 之内任意元素乘积在 32 位整数范围内。**请不要使用除法**，且需在 O(n) 时间、O(1) 额外空间（输出数组不计）内完成。

### 示例

```
输入：nums = [1,2,3,4]
输出：[24,12,8,6]

输入：nums = [-1,1,0,-3,3]
输出：[0,0,9,0,0]
```

### 约束

- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`
- 保证乘积在 32 位整数范围内

---

## 解法一：前缀积 + 后缀积（推荐）

### 思路

`answer[i] = 左边所有乘积 × 右边所有乘积`。第一遍从左到右累乘前缀，第二遍从右到左累乘后缀。

### 代码

```python
from typing import List


class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [1] * n
        prefix = 1
        for i in range(n):
            ans[i] = prefix
            prefix *= nums[i]
        suffix = 1
        for i in range(n - 1, -1, -1):
            ans[i] *= suffix
            suffix *= nums[i]
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) 额外 |

### 讲解

两次扫描，输出数组同时存前缀积，第二遍乘后缀积。满足不用除法、O(1) 额外空间。

## 解法二：左右数组

### 思路

分别建 `left[i]` 和 `right[i]` 数组，再相乘得答案。

### 代码

```python
from typing import List


class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        left = [1] * n
        right = [1] * n
        for i in range(1, n):
            left[i] = left[i - 1] * nums[i - 1]
        for i in range(n - 2, -1, -1):
            right[i] = right[i + 1] * nums[i + 1]
        return [left[i] * right[i] for i in range(n)]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

思路最直观，但额外 O(n) 空间，不满足 follow-up 要求。

## 解法三：除法（不满足题意）

### 思路

先算总乘积，再除以 `nums[i]`。有 0 时需特殊处理。

### 代码

```python
from typing import List


class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        zero_count = nums.count(0)
        if zero_count > 1:
            return [0] * len(nums)
        total = 1
        for x in nums:
            if x != 0:
                total *= x
        ans = []
        for x in nums:
            if zero_count == 1:
                ans.append(total if x == 0 else 0)
            else:
                ans.append(total // x)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

题目明确禁止除法，此写法仅作对比，面试勿用。

## 解法对比

| 解法 | 时间 | 额外空间 | 适用场景 |
|------|------|----------|----------|
| 前缀+后缀一次 | O(n) | O(1) | 面试首选 |
| 左右数组 | O(n) | O(n) | 理解过渡 |
| 除法 | O(n) | O(1) | 不满足题意 |

---

## 常见错误

1. **直接用除法**：题目禁止，且有 0 时出错。
2. **后缀积循环方向错**：应从 n-1 到 0。
3. **忘记输出数组不计入空间**：解法一符合 O(1) 额外空间。
