# 最大子数组和（Maximum Subarray）

- 题目链接：https://leetcode.cn/problems/maximum-subarray/
- 难度：中等
- 标签：数组、分治、动态规划

## 题目描述

给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组** 是数组中的一个连续部分。

### 示例

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6。

输入：nums = [1]
输出：1

输入：nums = [5,4,-1,7,8]
输出：23
```

### 约束

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

---

## 解法一：暴力枚举（最容易想到）

### 思路

枚举所有子数组 `[i, j]`，计算区间和，取最大值。

### 代码

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        ans = nums[0]
        for i in range(n):
            cur = 0
            for j in range(i, n):
                cur += nums[j]
                ans = max(ans, cur)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(1) |

### 讲解

暴力法能验证「子数组」定义是否正确，但 `n = 10^5` 时会超时。可作为理解题意的起点。

---

## 解法二：前缀和优化暴力

### 思路

固定左端点 `i`，用前缀和 `prefix[j] - prefix[i-1]` 快速算 `[i, j]` 的和，仍是 O(n²)，但常数更小。

### 代码

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        n = len(nums)
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]

        ans = nums[0]
        for i in range(n):
            for j in range(i, n):
                cur = prefix[j + 1] - prefix[i]
                ans = max(ans, cur)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(n) |

## 解法三：Kadane 算法（推荐，最优）

### 思路

定义 `cur`：**以当前元素结尾**的最大子数组和。

- 若 `cur + nums[i] < nums[i]`，说明前面的累加是负贡献，不如从 `nums[i]` 重新开始。
- 因此：`cur = max(nums[i], cur + nums[i])`
- 全局答案：`ans = max(ans, cur)`

### 代码

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        cur = ans = nums[0]
        for num in nums[1:]:
            cur = max(num, cur + num)
            ans = max(ans, cur)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 图解（nums = [-2,1,-3,4,-1,2,1,-5,4]）

```
i=0: cur=-2, ans=-2
i=1: cur=1,  ans=1
i=2: cur=-2, ans=1
i=3: cur=4,  ans=4
i=4: cur=3,  ans=4
i=5: cur=5,  ans=5
i=6: cur=6,  ans=6  ← 最终答案
```

### 正确性直觉

任意最大子数组 `[l, r]`，当我们遍历到 `r` 时，`cur` 恰好是以 `r` 结尾的最优和；若最优子数组在更早位置结束，`ans` 已记录。

## 解法四：分治（了解即可）

将数组一分为二，最大子数组要么在左半、右半，要么**跨越中点**。跨越中点的部分从中点向两侧各扩展一次，整体 T(n) = 2T(n/2) + O(n) → O(n log n)。

面试中 Kadane 更常用；分治用于理解「最大子数组」的结构性。

### 代码

```python
from typing import List


class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        def cross_max(lo: int, mid: int, hi: int) -> int:
            left_sum = float("-inf")
            cur = 0
            for i in range(mid, lo - 1, -1):
                cur += nums[i]
                left_sum = max(left_sum, cur)
            right_sum = float("-inf")
            cur = 0
            for i in range(mid + 1, hi + 1):
                cur += nums[i]
                right_sum = max(right_sum, cur)
            return left_sum + right_sum

        def divide(lo: int, hi: int) -> int:
            if lo == hi:
                return nums[lo]
            mid = (lo + hi) // 2
            return max(divide(lo, mid), divide(mid + 1, hi), cross_max(lo, mid, hi))

        return divide(0, len(nums) - 1)
```

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 暴力枚举 | O(n²) | O(1) | 理解题意 |
| Kadane | O(n) | O(1) | 面试与提交首选 |
| 分治 | O(n log n) | O(log n) | 算法课、拓展 |

## 常见错误

1. **子数组 vs 子序列**：必须连续，不能跳着选。
2. **全负数数组**：至少选一个元素，答案为最大单个元素（如 `[-3,-1,-2]` → `-1`）。
3. **初始化**：`cur` 和 `ans` 应从 `nums[0]` 开始，而非 0。

---

## 举一反三

- **乘积最大子数组**：维护最大/最小乘积（负数翻转）
- **环形子数组的最大和**：Kadane + 总和减去最小子数组
- **股票买卖**：状态机 DP，思想类似「延续或重新开始」
