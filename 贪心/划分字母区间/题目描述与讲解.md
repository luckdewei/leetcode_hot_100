# 划分字母区间（Partition Labels）

- 题目链接：https://leetcode.cn/problems/partition-labels/
- 难度：中等
- 标签：贪心、哈希表、双指针

## 题目描述

字符串 `s` 由小写字母组成。要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。

返回一个列表，表示每个字符串片段的长度。

### 示例

```
输入：s = "ababcbacadefegdehijhklij"
输出：[9,7,8]
解释：划分 "ababcbaca"、"defegde"、"hijhklij"。

输入：s = "eccbbbbdec"
输出：[10]
```

### 约束

- `1 <= s.length <= 500`
- `s` 仅由小写英文字母组成

---

## 解法一：记录最后出现位置 + 贪心（推荐）

### 思路

预处理每个字母最后出现下标。遍历时扩展当前片段右边界 `end` 为各字符最后位置的最大值；当 `i == end` 时切分一段。

### 代码

```python
from typing import List


class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        last = {ch: i for i, ch in enumerate(s)}
        ans = []
        start = end = 0
        for i, ch in enumerate(s):
            end = max(end, last[ch])
            if i == end:
                ans.append(end - start + 1)
                start = i + 1
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1)（字母表大小 26） |

### 讲解

贪心正确性：当前片段必须覆盖片段内所有字母的最后出现位置，否则同一字母会跨片段。

## 解法二：合并区间

### 思路

每个字母对应区间 `[first, last]`，合并所有重叠区间，区间长度即片段长度。

### 代码

```python
from typing import List


class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        bounds = {}
        for i, ch in enumerate(s):
            if ch not in bounds:
                bounds[ch] = [i, i]
            else:
                bounds[ch][1] = i
        intervals = sorted(bounds.values())
        merged = [intervals[0]]
        for start, end in intervals[1:]:
            if start <= merged[-1][1]:
                merged[-1][1] = max(merged[-1][1], end)
            else:
                merged.append([start, end])
        return [e - s + 1 for s, e in merged]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与解法一等价，转化为区间合并问题。字母只有 26 种，排序开销可忽略。

## 解法三：暴力扩展（理解用）

### 思路

从左到右，每次尽量扩展当前片段直到不包含外部分割字母。

### 代码

```python
from typing import List


class Solution:
    def partitionLabels(self, s: str) -> List[int]:
        ans = []
        i = 0
        n = len(s)
        while i < n:
            end = i
            for j in range(i, end + 1):
                end = max(end, s.rfind(s[j]))
            ans.append(end - i + 1)
            i = end + 1
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) 最坏 |
| 空间 | O(1) |

### 讲解

内层 `rfind` 导致 O(n²)，预处理 last 数组可优化为 O(n)。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 贪心+last | O(n) | O(1) | 面试首选 |
| 区间合并 | O(n) | O(1) | 等价理解 |
| 暴力扩展 | O(n²) | O(1) | 理解用 |

---

## 常见错误

1. **用 first 而非 last 扩展边界**：应取最后出现位置。
2. **切分条件写错**：`i == end` 时才能切，不是 `i > end`。
3. **与「合并区间」混淆**：本题区间由字母自然产生。
