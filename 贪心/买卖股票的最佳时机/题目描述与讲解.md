# 买卖股票的最佳时机（Best Time to Buy and Sell Stock）

- 题目链接：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/
- 难度：简单
- 标签：数组、贪心、动态规划

## 题目描述

给定数组 `prices`，`prices[i]` 是第 i 天股票价格。你只能选择**某一天买入、未来某一天卖出**，计算能获取的最大利润。若不能获利返回 0。

### 示例

```
输入：prices = [7,1,5,3,6,4]
输出：5
解释：第 2 天买(1)，第 5 天卖(6)，利润 5。

输入：prices = [7,6,4,3,1]
输出：0
```

### 约束

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

---

## 解法一：一次遍历贪心（推荐）

### 思路

维护历史最低买入价 `min_price`，每天更新 `max(profit, price - min_price)`。

### 代码

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_price = float("inf")
        ans = 0
        for p in prices:
            min_price = min(min_price, p)
            ans = max(ans, p - min_price)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

等价于「在后面找最大值减去前面最小值」，但一次扫描即可。必须先买后卖。

## 解法二：动态规划

### 思路

`dp[i]` 表示第 i 天卖出时的最大利润；或维护 `min_so_far` 和 `max_profit` 两个状态。

### 代码

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if not prices:
            return 0
        min_price = prices[0]
        max_profit = 0
        for i in range(1, len(prices)):
            max_profit = max(max_profit, prices[i] - min_price)
            min_price = min(min_price, prices[i])
        return max_profit
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与贪心等价，状态转移：`max_profit = max(max_profit, price - min_price)`。

## 解法三：暴力枚举（理解用）

### 思路

枚举所有 (买入日, 卖出日) 对，取最大利润。

### 代码

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        ans = 0
        for i in range(n):
            for j in range(i + 1, n):
                ans = max(ans, prices[j] - prices[i])
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(1) |

### 讲解

O(n²) 会超时，帮助理解题意。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 一次遍历 | O(n) | O(1) | 面试首选 |
| DP | O(n) | O(1) | 等价写法 |
| 暴力 | O(n²) | O(1) | 理解用 |

---

## 常见错误

1. **允许先卖后买**：题目只能先买后卖。
2. **与 II 题混淆**：II 题可多次交易，本题只能一次。
3. **价格下跌仍更新 min**：应持续维护最低买入价。
