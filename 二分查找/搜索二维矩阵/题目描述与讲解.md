# 搜索二维矩阵（Search a 2D Matrix）

- 题目链接：https://leetcode.cn/problems/search-a-2d-matrix/
- 难度：中等
- 标签：数组、二分查找、矩阵

## 题目描述

给你一个满足下述两条属性的 `m x n` 整数矩阵：

- 每行元素从左到右升序排列
- 每行第一个整数大于上一行最后一个整数

给定一个整数 `target`，判断 `target` 是否在该矩阵中。

### 示例

```
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true

输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
输出：false
```

### 约束

- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= m, n <= 100`
- `-10^4 <= matrix[i][j], target <= 10^4`

---

## 解法一：逐行二分

### 思路

矩阵每行有序，对每一行做二分查找，找到即返回 True。

### 代码

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        for row in matrix:
            left, right = 0, len(row) - 1
            while left <= right:
                mid = (left + right) // 2
                if row[mid] == target:
                    return True
                elif row[mid] < target:
                    left = mid + 1
                else:
                    right = mid - 1
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m log n) |
| 空间 | O(1) |

## 解法二：展平为一维数组二分（推荐）

### 思路

题目保证「每行首元素大于上一行末元素」，等价于整个矩阵按行展开后是一维升序数组。

对下标 `mid`（0 到 m×n-1）：
- 行号：`mid // n`
- 列号：`mid % n`

### 代码

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False
        m, n = len(matrix), len(matrix[0])
        left, right = 0, m * n - 1
        while left <= right:
            mid = (left + right) // 2
            val = matrix[mid // n][mid % n]
            if val == target:
                return True
            elif val < target:
                left = mid + 1
            else:
                right = mid - 1
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(log(mn)) |
| 空间 | O(1) |

### 讲解

一次二分即可，比逐行二分更优。核心是「二维下标 ↔ 一维下标」的映射。

## 解法三：从右上角走 Z 字形（非二分，拓展）

### 思路

从右上角 `(0, n-1)` 出发：当前值大于 target 则左移，小于 target 则下移。

适用于**每行、每列分别有序**的变体（LeetCode 240），本题同样可用但不如二分简洁。

### 代码

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False
        row, col = 0, len(matrix[0]) - 1
        while row < len(matrix) and col >= 0:
            if matrix[row][col] == target:
                return True
            elif matrix[row][col] > target:
                col -= 1
            else:
                row += 1
        return False
```

---

## 解法对比

| 解法 | 时间 | 空间 | 说明 |
|------|------|------|------|
| 逐行二分 | O(m log n) | O(1) | 直观 |
| 一维二分 | O(log(mn)) | O(1) | 最优 |
| Z 形搜索 | O(m+n) | O(1) | 更通用矩阵搜索 |

---

## 常见错误

1. **行列映射错误**：`matrix[mid // n][mid % n]` 中 m、n 不要写反。
2. **忽略空矩阵**：`matrix` 或 `matrix[0]` 为空时要提前返回。
3. **误用 240 题解法**：本题行间也有序，应优先一维二分。
