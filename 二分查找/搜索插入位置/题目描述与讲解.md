# 搜索插入位置（Search Insert Position）

- 题目链接：https://leetcode.cn/problems/search-insert-position/
- 难度：简单
- 标签：数组、二分查找

## 题目描述

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 O(log n) 的算法。

### 示例

```
输入：nums = [1,3,5,6], target = 5
输出：2

输入：nums = [1,3,5,6], target = 2
输出：1

输入：nums = [1,3,5,6], target = 7
输出：4

输入：nums = [1,3,5,6], target = 0
输出：0
```

### 约束

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 为**无重复元素**的升序排列数组
- `-10^4 <= target <= 10^4`

---

## 解法一：线性扫描

### 思路

从左到右遍历，找到第一个 `>= target` 的位置；若全部小于 target，返回 `len(nums)`。

### 代码

```python
from typing import List


class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        for i, num in enumerate(nums):
            if num >= target:
                return i
        return len(nums)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

直观但不符合题目 O(log n) 要求，适合理解「插入位置」的含义。

---

## 解法二：标准二分查找（推荐）

### 思路

在有序数组中查找 target。若找到直接返回下标；若未找到，循环结束时 `left` 即为插入位置。

**关键理解：** 二分结束后，`left` 指向第一个大于 target 的位置（或 `len(nums)`）。

### 代码

```python
from typing import List


class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        return left
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(log n) |
| 空间 | O(1) |

### 讲解

本题是二分查找的入门题，与「在排序数组中查找元素的第一个和最后一个位置」一脉相承。

**边界情况：**
- target 比所有元素都小 → 返回 0
- target 比所有元素都大 → 返回 `len(nums)`
- target 恰好在中间 → 正常返回 mid

## 解法三：左边界二分（lower_bound 模板）

### 思路

找第一个 `>= target` 的下标，即 C++ `lower_bound` 语义。

```python
left, right = 0, len(nums)  # 左闭右开
while left < right:
    mid = (left + right) // 2
    if nums[mid] < target:
        left = mid + 1
    else:
        right = mid
return left
```

### 讲解

左闭右开写法更统一，便于扩展到「第一个等于」「第一个大于」等变体。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 线性扫描 | O(n) | O(1) | 理解题意 |
| 标准二分 | O(log n) | O(1) | 面试首选 |
| lower_bound | O(log n) | O(1) | 模板化练习 |

---

## 常见错误

1. **循环条件写错**：`left <= right` 与 `left < right` 对应不同边界处理。
2. **返回 right 而非 left**：在 `left <= right` 写法中应返回 `left`。
3. **mid 溢出**：Python 整数无溢出，其他语言可用 `left + (right - left) // 2`。
