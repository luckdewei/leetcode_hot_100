# 字母异位词分组（Group Anagrams）

- 题目链接：https://leetcode.cn/problems/group-anagrams/
- 难度：中等
- 标签：数组、哈希表、字符串、排序

## 题目描述

给你一个字符串数组，请你将**字母异位词**组合在一起。可以按任意顺序返回结果列表。

**字母异位词**是由重新排列源单词的所有字母得到的单词。

### 示例

```
输入：strs = ["eat","tea","tan","ate","nat","bat"]
输出：[["bat"],["nat","tan"],["ate","eat","tea"]]

输入：strs = [""]
输出：[[""]]

输入：strs = ["a"]
输出：[["a"]]
```

### 约束

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅由小写英文字母组成

---

## 解法一：排序后作为 key（推荐）

### 思路

两个字符串是异位词，当且仅当它们**排序后完全相同**。

因此可以把每个字符串排序后的结果作为哈希表的 key，把原字符串归入对应分组。

### 代码

```python
from collections import defaultdict
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)
        for s in strs:
            key = tuple(sorted(s))
            groups[key].append(s)
        return list(groups.values())
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × k log k)，n 为字符串个数，k 为单个字符串最大长度 |
| 空间 | O(n × k)，存储所有字符串及分组 |

### 讲解

这是最直接、面试最常用的写法。

**为什么排序 key 有效？**  
异位词拥有相同的字符 multiset。排序后得到相同的字符序列，例如 `"eat"`、`"tea"`、`"ate"` 排序后都是 `"aet"`。

**实现细节：**

- 用 `defaultdict(list)` 自动建组
- key 用 `tuple(sorted(s))` 而非字符串，因为列表不能作为 dict key
- 最后 `list(groups.values())` 返回分组列表

## 解法二：字符频次计数作为 key

### 思路

不排序，而是统计每个字符串中 26 个小写字母的出现次数，把频次元组作为 key。

例如 `"eat"` → `(1,0,0,0,1,0,...,1,0,...)`（a=1, e=1, t=1，其余为 0）。

### 代码

```python
from collections import defaultdict
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)
        for s in strs:
            count = [0] * 26
            for ch in s:
                count[ord(ch) - ord('a')] += 1
            key = tuple(count)
            groups[key].append(s)
        return list(groups.values())
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × k)，每个字符只遍历一次 |
| 空间 | O(n × k) |

### 讲解

当单个字符串很长时，计数法比排序略优（避免 log k 因子）。

**优点**：时间更优，适合 k 较大的场景。  
**缺点**：key 是长度 26 的元组，常数略大；代码比排序法稍长。

## 解法三：暴力两两比较（理解用）

### 思路

枚举每对字符串，用排序或 Counter 判断是否互为异位词，用并查集或标记合并同组。

### 代码

```python
from typing import List


class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        n = len(strs)
        parent = list(range(n))

        def find(x: int) -> int:
            while parent[x] != x:
                parent[x] = parent[parent[x]]
                x = parent[x]
            return x

        def union(a: int, b: int) -> None:
            pa, pb = find(a), find(b)
            if pa != pb:
                parent[pa] = pb

        for i in range(n):
            for j in range(i + 1, n):
                if sorted(strs[i]) == sorted(strs[j]):
                    union(i, j)

        groups = {}
        for i in range(n):
            root = find(i)
            groups.setdefault(root, []).append(strs[i])
        return list(groups.values())
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n² × k log k) |
| 空间 | O(n) |

### 讲解

仅用于理解题意，数据规模下会超时，不推荐提交。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 排序 key | O(n × k log k) | O(n × k) | 面试首选，简洁 |
| 频次 key | O(n × k) | O(n × k) | k 很大时略优 |
| 暴力比较 | O(n² × k log k) | O(n) | 理解题意 |

## 常见错误

1. **key 用了 list 而不是 tuple**：Python 中 list 不可哈希，不能作为 dict key。
2. **忘记处理空字符串**：`""` 排序后仍是 `""`，应正常分组。
3. **返回顺序纠结**：题目允许任意顺序，分组内、分组间都不需要排序（除非题目额外要求）。
4. **与「找到字符串中所有字母异位词」混淆**：本题是**分组**，438 题是在主串中**找**异位词子串的起始下标。
