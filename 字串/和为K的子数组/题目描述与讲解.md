# 和为 K 的子数组（Subarray Sum Equals K）

- 题目链接：https://leetcode.cn/problems/subarray-sum-equals-k/
- 难度：中等
- 标签：数组、哈希表、前缀和

## 题目描述

给你一个整数数组 `nums` 和一个整数 `k`，请你统计并返回该数组中和为 `k` 的**连续子数组**的个数。

### 示例

```
输入：nums = [1,1,1], k = 2
输出：2

输入：nums = [1,2,3], k = 3
输出：2
```

### 约束

- `1 <= nums.length <= 2 * 10^4`
- `-1000 <= nums[i] <= 1000`
- `-10^7 <= k <= 10^7`

---

## 解法一：前缀和 + 哈希表（推荐）

### 思路

若子数组 `[j+1..i]` 和为 k，则 `prefix[i] - prefix[j] = k`，即 `prefix[j] = prefix[i] - k`。遍历时用哈希表记录各前缀和出现次数。

### 代码

```python
from collections import defaultdict
from typing import List


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        prefix_count = defaultdict(int)
        prefix_count[0] = 1
        prefix = 0
        ans = 0
        for num in nums:
            prefix += num
            ans += prefix_count[prefix - k]
            prefix_count[prefix] += 1
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

`prefix_count[0] = 1` 处理从开头开始的子数组。注意数组含负数，不能用滑动窗口，必须用前缀和。

## 解法二：暴力枚举

### 思路

枚举所有子数组起点和终点，计算区间和，等于 k 则计数。

### 代码

```python
from typing import List


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        n = len(nums)
        ans = 0
        for i in range(n):
            s = 0
            for j in range(i, n):
                s += nums[j]
                if s == k:
                    ans += 1
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(1) |

### 讲解

直观但 n=2×10⁴ 时会超时，仅用于理解。

## 解法三：前缀和数组 + 枚举

### 思路

先 O(n) 建前缀和数组，再 O(n²) 枚举所有 `(i, j)` 对检查 `prefix[j+1]-prefix[i]`。

### 代码

```python
from typing import List


class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        n = len(nums)
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]
        ans = 0
        for i in range(n):
            for j in range(i, n):
                if prefix[j + 1] - prefix[i] == k:
                    ans += 1
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(n) |

### 讲解

比暴力略清晰，但仍 O(n²)。哈希表法是一次遍历的优化。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 前缀和+哈希 | O(n) | O(n) | 面试首选 |
| 暴力枚举 | O(n²) | O(1) | 理解用 |
| 前缀和枚举 | O(n²) | O(n) | 过渡理解 |

## 常见错误

1. **忘记初始化 `prefix_count[0]=1`**：漏计从 index 0 开始的子数组。
2. **用滑动窗口**：数组有负数时窗口和不一定单调。
3. **与「两数之和」混淆**：本题求连续子数组个数，不是两数配对。
