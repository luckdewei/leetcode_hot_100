# 排序链表（Sort List）

- 题目链接：https://leetcode.cn/problems/sort-list/
- 难度：中等
- 标签：链表、双指针、分治、排序、归并排序

## 题目描述

给你链表的头结点 `head`，请将其按升序排列并返回。要求 O(n log n) 时间、O(1) 空间（递归栈不计入则近似满足）。

### 示例

```
输入：head = [4,2,1,3]
输出：[1,2,3,4]

输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]

输入：head = []
输出：[]
```

### 约束

- 节点数 `[0, 5 * 10^4]`
- `-10^5 <= Node.val <= 10^5`

---

## 解法一：归并排序（推荐）

### 思路

快慢指针找中点，断开两半递归排序，再合并两个有序链表。与数组归并排序相同框架。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        mid = slow.next
        slow.next = None
        left = self.sortList(head)
        right = self.sortList(mid)
        return self._merge(left, right)

    def _merge(self, l1, l2):
        dummy = ListNode(0)
        cur = dummy
        while l1 and l2:
            if l1.val <= l2.val:
                cur.next = l1
                l1 = l1.next
            else:
                cur.next = l2
                l2 = l2.next
            cur = cur.next
        cur.next = l1 if l1 else l2
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n log n) |
| 空间 | O(log n) 递归栈 |

### 讲解

链表归并排序标准写法。`fast = head.next` 保证中点划分正确，避免无限递归。

## 解法二：转数组 + 排序

### 思路

值存入数组，排序后重建链表。

### 代码

```python
from typing import Optional, List


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        vals: List[int] = []
        while head:
            vals.append(head.val)
            head = head.next
        vals.sort()
        dummy = ListNode(0)
        cur = dummy
        for v in vals:
            cur.next = ListNode(v)
            cur = cur.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n log n) |
| 空间 | O(n) |

### 讲解

不满足 O(1) 空间，但实现简单，可用于验证。

## 解法三：插入排序（理解用）

### 思路

逐个节点插入已排序部分的正确位置。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        cur = head
        while cur:
            prev = dummy
            while prev.next and prev.next.val < cur.val:
                prev = prev.next
            nxt = cur.next
            cur.next = prev.next
            prev.next = cur
            cur = nxt
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(1) |

### 讲解

O(n²) 会超时，帮助理解链表插入操作。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 归并排序 | O(n log n) | O(log n) | 面试首选 |
| 数组排序 | O(n log n) | O(n) | 快速验证 |
| 插入排序 | O(n²) | O(1) | 理解用 |

---

## 常见错误

1. **中点划分无限递归**：`slow.next = None` 必须断开。
2. **fast 初值错**：应 `fast = head.next` 找上中点。
3. **用冒泡交换值**：题目要求 O(n log n)。
