# 相交链表（Intersection of Two Linked Lists）

- 题目链接：https://leetcode.cn/problems/intersection-of-two-linked-lists/
- 难度：简单
- 标签：哈希表、链表、双指针

## 题目描述

给你两个单链表的头节点 `headA` 和 `headB`，找出并返回两个单链表相交的起始节点。若不相交返回 `null`。

相交是指两链表共享部分节点（同一节点对象引用，非 merely 值相等）。

### 示例

```
输入：intersectVal = 8, listA = [4,1,8,4,5], listB = [5,6,1,8,4,5]
输出：Intersected at '8'

输入：intersectVal = 2, listA = [1,9,1,2,4], listB = [3,2,4]
输出：Intersected at '2'

输入：intersectVal = 0, listA = [2,6,4], listB = [1,5]
输出：null
```

### 约束

- `listA`、`listB` 无环
- 节点数 `[0, 3 * 10^4]`
- `-10^5 <= Node.val <= 10^5`

---

## 解法一：双指针切换链表（推荐）

### 思路

指针 A 走完 A 后走 B，指针 B 走完 B 后走 A。两者走过的总长度相同，若有交点必同时到达交点；若无交点同时到 NULL。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        pa, pb = headA, headB
        while pa is not pb:
            pa = pa.next if pa else headB
            pb = pb.next if pb else headA
        return pa
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m + n) |
| 空间 | O(1) |

### 讲解

巧妙对齐路径长度差。A 长 a、B 长 b、公共尾 c，则 a+c = b+c 时相遇。

## 解法二：哈希集合

### 思路

遍历 A 所有节点入 set，再遍历 B 找第一个在 set 中的节点。

### 代码

```python
from typing import Optional, Set


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        seen: Set[ListNode] = set()
        while headA:
            seen.add(headA)
            headA = headA.next
        while headB:
            if headB in seen:
                return headB
            headB = headB.next
        return None
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m + n) |
| 空间 | O(m) |

### 讲解

直观，O(m) 额外空间。

## 解法三：双指针对齐长度

### 思路

先分别求 A、B 长度，长链表指针先走差值步，再同步走找交点。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        def length(head):
            n = 0
            while head:
                n += 1
                head = head.next
            return n

        la, lb = length(headA), length(headB)
        pa, pb = headA, headB
        while la > lb:
            pa = pa.next
            la -= 1
        while lb > la:
            pb = pb.next
            lb -= 1
        while pa is not pb:
            pa = pa.next
            pb = pb.next
        return pa
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m + n) |
| 空间 | O(1) |

### 讲解

与切换链表等价，需两次遍历求长度。

| 切换链表 | O(m+n) | O(1) | 面试首选 |

| 对齐长度 | O(m+n) | O(1) | 等价理解 |

## 常见错误

1. **比较值而非节点引用**：应用 `is` 比较同一对象。
2. **认为交点之后链表分叉**：相交是合并，不是分叉。
3. **切换条件写错**：`pa = pa.next if pa else headB`。
