# 环形链表 II（Linked List Cycle II）

- 题目链接：https://leetcode.cn/problems/linked-list-cycle-ii/
- 难度：中等
- 标签：哈希表、链表、双指针

## 题目描述

给定一个链表，返回链表开始入环的第一个节点。若不存在环，返回 `null`。不允许修改链表。

### 示例

```
输入：head = [3,2,0,-4], pos = 1
输出：返回下标 1 的节点（值为 2）

输入：head = [1,2], pos = 0
输出：返回下标 0 的节点

输入：head = [1], pos = -1
输出：null
```

### 约束

- 节点数 `[0, 10^4]`
- `-10^5 <= Node.val <= 10^5`
- `pos` 为 -1 或链表环入口下标

---

## 解法一：Floyd 找环入口（推荐）

### 思路

快慢指针相遇后，一新指针从 head 出发，与 slow 同步走一步，相遇点即为环入口。数学证明：head 到入口距离 = 相遇点到入口距离。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow is fast:
                ptr = head
                while ptr is not slow:
                    ptr = ptr.next
                    slow = slow.next
                return ptr
        return None
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

Phase 1 判圈，Phase 2 找入口。两指针各走入口前段长度即可相遇于入口。

## 解法二：哈希集合

### 思路

遍历时节点入 set，第一个重复出现的节点即环入口。

### 代码

```python
from typing import Optional, Set


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        seen: Set[ListNode] = set()
        while head:
            if head in seen:
                return head
            seen.add(head)
            head = head.next
        return None
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

第一个重复节点即环入口，简单但 O(n) 空间。

## 解法三：标记法（不推荐）

### 思路

遍历时修改节点值或属性标记已访问，第一个被标记的即入口。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None


class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        MARK = float("inf")
        while head:
            if head.val == MARK:
                return head
            head.val = MARK
            head = head.next
        return None
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

修改了链表，违反题意「不允许修改」，仅作反面教材。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| Floyd 双阶段 | O(n) | O(1) | 面试首选 |
| 哈希集合 | O(n) | O(n) | 直观写法 |
| 标记法 | O(n) | O(1) | 不推荐 |

## 常见错误

1. **相遇后直接返回 slow**：需第二阶段从 head 同步走。
2. **Phase 2 只动一个指针**：ptr 和 slow 应同步各走一步。
3. **与 141 题混淆**：141 只判有无环，142 要找入口。
