# 两两交换链表中的节点（Swap Nodes in Pairs）

- 题目链接：https://leetcode.cn/problems/swap-nodes-in-pairs/
- 难度：中等
- 标签：递归、链表

## 题目描述

给你一个链表，两两交换相邻节点，返回交换后链表的头节点。必须修改节点指针，不能仅修改值。

### 示例

```
输入：head = [1,2,3,4]
输出：[2,1,4,3]

输入：head = []
输出：[]

输入：head = [1]
输出：[1]
```

### 约束

- 节点数 `[0, 100]`
- `0 <= Node.val <= 100`

---

## 解法一：迭代 + 哑节点（推荐）

### 思路

哑节点 `dummy`，`prev` 指向待交换对的前一个节点。每次交换 `prev.next` 与 `prev.next.next`，然后 `prev` 移到交换后的第二个节点。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        prev = dummy
        while prev.next and prev.next.next:
            first = prev.next
            second = first.next
            first.next = second.next
            second.next = first
            prev.next = second
            prev = first
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

注意指针修改顺序，避免断链。奇数个节点时最后一组不交换。

## 解法二：递归

### 思路

交换前两个节点，将第一个的 next 指向后续递归结果。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        new_head = head.next
        head.next = self.swapPairs(new_head.next)
        new_head.next = head
        return new_head
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) 递归栈 |

### 讲解

代码简洁，但栈空间 O(n/2)。面试迭代更常考。

## 解法三：转数组重建（理解用）

### 思路

值存入数组，两两交换后重建链表。

### 代码

```python
from typing import Optional, List


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        vals: List[int] = []
        while head:
            vals.append(head.val)
            head = head.next
        for i in range(0, len(vals) - 1, 2):
            vals[i], vals[i + 1] = vals[i + 1], vals[i]
        dummy = ListNode(0)
        cur = dummy
        for v in vals:
            cur.next = ListNode(v)
            cur = cur.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

题目要求修改指针，此写法不满足要求。

## 常见错误

1. **指针顺序错导致断链**：先存 second.next 再改 first.next。
2. **忘记哑节点**：头节点交换后 head 会变。
3. **与 K 组翻转混淆**：25 题是 k 个一组通用版。
