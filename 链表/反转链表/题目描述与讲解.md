# 反转链表（Reverse Linked List）

- 题目链接：https://leetcode.cn/problems/reverse-linked-list/
- 难度：简单
- 标签：递归、链表

## 题目描述

给你单链表的头节点 `head`，反转链表并返回反转后的头节点。

### 示例

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]

输入：head = [1,2]
输出：[2,1]

输入：head = []
输出：[]
```

### 约束

- 节点数 `[0, 5000]`
- `-5000 <= Node.val <= 5000`

---

## 解法一：迭代三指针（推荐）

### 思路

维护 `prev`、`cur`、`nxt`，逐个反转 `cur.next` 指向 `prev`，然后三指针前移。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        prev = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        return prev
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

链表反转基础题，必须熟练。返回 `prev` 即新头节点。

---

## 解法二：递归

### 思路

递归反转 `head.next` 之后的链表，令 `head.next.next = head`，`head.next = None`。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return head
        new_head = self.reverseList(head.next)
        head.next.next = head
        head.next = None
        return new_head
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) 递归栈 |

### 讲解

递归版简洁，但栈深度 O(n)。面试两种都应会。

---

## 解法三：头插法

### 思路

遍历原链表，每次将当前节点插入新链表头部。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        new_head = None
        while head:
            nxt = head.next
            head.next = new_head
            new_head = head
            head = nxt
        return new_head
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与迭代三指针等价，只是变量语义不同（构建新头 vs 原地反转）。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 迭代 | O(n) | O(1) | 面试首选 |
| 递归 | O(n) | O(n) | 简洁写法 |
| 头插法 | O(n) | O(1) | 等价理解 |

---

## 常见错误

1. **忘记 `head.next = None`**：递归版不置空会形成环。
2. **返回 head 而非 prev**：迭代结束 cur 为 None，新头是 prev。
3. **断链顺序错**：必须先存 nxt 再改 cur.next。
