# 两数相加（Add Two Numbers）

- 题目链接：https://leetcode.cn/problems/add-two-numbers/
- 难度：中等
- 标签：递归、链表、数学

## 题目描述

给你两个**非空**链表，每位存一位数字（**逆序**存储），即个位在链表头部。将两数相加，返回相同形式表示和的链表。

假设除 0 外不含前导零。

### 示例

```
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807

输入：l1 = [0], l2 = [0]
输出：[0]

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

### 约束

- 每个链表节点数 `[1, 100]`
- `0 <= Node.val <= 9`
- 不含前导零

---

## 解法一：模拟竖式加法（推荐）

### 思路

同时遍历两链表，维护进位 `carry`。每位 `s = carry + l1.val + l2.val`，`digit = s % 10`，`carry = s // 10`。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        cur = dummy
        carry = 0
        while l1 or l2 or carry:
            s = carry
            if l1:
                s += l1.val
                l1 = l1.next
            if l2:
                s += l2.val
                l2 = l2.next
            carry, digit = divmod(s, 10)
            cur.next = ListNode(digit)
            cur = cur.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(max(m,n)) |
| 空间 | O(1) 额外（不含输出） |

### 讲解

循环条件 `l1 or l2 or carry` 处理最后进位。逆序存储使加法自然从左（个位）到右。

## 解法二：递归

### 思路

递归处理下一位，当前位 `(l1.val + l2.val + carry) % 10`。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode], carry: int = 0) -> Optional[ListNode]:
        if not l1 and not l2 and carry == 0:
            return None
        s = carry
        if l1:
            s += l1.val
            l1 = l1.next
        if l2:
            s += l2.val
            l2 = l2.next
        node = ListNode(s % 10)
        node.next = self.addTwoNumbers(l1, l2, s // 10)
        return node
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(max(m,n)) |
| 空间 | O(max(m,n)) 递归栈 |

### 讲解

逻辑与迭代相同，递归栈可能较深。

## 解法三：转整数再转回（理解用）

### 思路

链表转整数相加，再转回链表。大数会溢出。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        def to_int(node):
            num, base = 0, 1
            while node:
                num += node.val * base
                base *= 10
                node = node.next
            return num

        total = to_int(l1) + to_int(l2)
        dummy = ListNode(0)
        cur = dummy
        if total == 0:
            return ListNode(0)
        while total:
            cur.next = ListNode(total % 10)
            total //= 10
            cur = cur.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(max(m,n)) |
| 空间 | O(max(m,n)) |

### 讲解

长链表会溢出，不满足题意，仅帮助理解。

## 常见错误

1. **忘记最后 carry**：循环条件需含 `carry`。
2. **与「两数相加 II」混淆**：445 题正序存储需反转。
3. **修改原链表 vs 新建**：通常新建结果节点。
