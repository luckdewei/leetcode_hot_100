# 随机链表的复制（Copy List with Random Pointer）

- 题目链接：https://leetcode.cn/problems/copy-list-with-random-pointer/
- 难度：中等
- 标签：哈希表、链表

## 题目描述

给定一个链表，每个节点包含额外随机指针 `random`，指向链表任意节点或 `null`。

构造该链表的**深拷贝**。深拷贝应完全独立，原链表与拷贝链表无共享节点。

### 示例

```
输入：head = [[7,null],[13,0],[11,4],[10,1],[1,0]]
输出：[[7,null],[13,0],[11,4],[10,1],[1,0]]

输入：head = [[1,1],[2,1]]
输出：[[1,1],[2,1]]

输入：head = [[3,null],[3,0],[3,null]]
输出：[[3,null],[3,0],[3,null]]
```

### 约束

- `0 <= n <= 1000`
- `-10^4 <= Node.val <= 10^4`
- `Node.random` 为 null 或指向链表内节点

---

## 解法一：哈希表映射（推荐）

### 思路

第一遍遍历创建所有新节点，建立 `old -> new` 映射；第二遍设置 `next` 和 `random` 指针。

### 代码

```python
from typing import Optional


class Node:
    def __init__(self, x: int, next: "Node" = None, random: "Node" = None):
        self.val = int(x)
        self.next = next
        self.random = random


class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        if not head:
            return None
        old_to_new = {None: None}
        cur = head
        while cur:
            old_to_new[cur] = Node(cur.val)
            cur = cur.next
        cur = head
        while cur:
            old_to_new[cur].next = old_to_new[cur.next]
            old_to_new[cur].random = old_to_new[cur.random]
            cur = cur.next
        return old_to_new[head]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

`{None: None}` 简化 random 为 null 的处理。两遍扫描，逻辑清晰。

## 解法二：交错节点 + 拆分

### 思路

在每个原节点后插入拷贝节点 A->A'->B->B'...；拷贝节点的 random = 原节点 random.next；最后拆分两链表。

### 代码

```python
from typing import Optional


class Node:
    def __init__(self, x: int, next: "Node" = None, random: "Node" = None):
        self.val = int(x)
        self.next = next
        self.random = random


class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        if not head:
            return None
        cur = head
        while cur:
            copy = Node(cur.val, cur.next)
            cur.next = copy
            cur = copy.next
        cur = head
        while cur:
            if cur.random:
                cur.next.random = cur.random.next
            cur = cur.next.next
        dummy = Node(0)
        copy_cur = dummy
        cur = head
        while cur:
            copy = cur.next
            cur.next = copy.next
            copy.next = cur.next.next if cur.next else None
            copy_cur.next = copy
            copy_cur = copy
            cur = cur.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) 额外 |

### 讲解

O(1) 额外空间的经典技巧。`random.next` 即对应拷贝节点。面试进阶写法。

## 解法三：递归 + 哈希（记忆化）

### 思路

递归复制节点，哈希表避免重复创建。

### 代码

```python
from typing import Optional, Dict


class Node:
    def __init__(self, x: int, next: "Node" = None, random: "Node" = None):
        self.val = int(x)
        self.next = next
        self.random = random


class Solution:
    def copyRandomList(self, head: Optional[Node]) -> Optional[Node]:
        memo: Dict[Node, Node] = {}

        def clone(node):
            if not node:
                return None
            if node in memo:
                return memo[node]
            copy = Node(node.val)
            memo[node] = copy
            copy.next = clone(node.next)
            copy.random = clone(node.random)
            return copy

        return clone(head)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

递归版哈希表，与迭代两遍等价，代码更紧凑。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 哈希两遍 | O(n) | O(n) | 面试首选 |
| 交错拆分 | O(n) | O(1) | 空间优化 |
| 递归哈希 | O(n) | O(n) | 简洁写法 |

---

## 常见错误

1. **浅拷贝**：只复制值，next/random 仍指向原节点。
2. **random 映射遗漏**：第二遍必须设置 random。
3. **交错法拆分断链**：拆分顺序要正确。
