# 回文链表（Palindrome Linked List）

- 题目链接：https://leetcode.cn/problems/palindrome-linked-list/
- 难度：简单
- 标签：栈、递归、链表、双指针

## 题目描述

给你一个单链表的头节点 `head`，请判断该链表是否为回文链表。要求 O(n) 时间、O(1) 空间（若允许 O(n) 空间则更简单）。

### 示例

```
输入：head = [1,2,2,1]
输出：true

输入：head = [1,2]
输出：false
```

### 约束

- 节点数 `[1, 10^5]`
- `0 <= Node.val <= 9`

---

## 解法一：找中点 + 反转后半 + 比较（推荐）

### 思路

快慢指针找中点，反转后半段，前后半段逐一比较。可选恢复链表（面试加分）。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        if not head or not head.next:
            return True
        slow = fast = head
        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next
        second = self._reverse(slow.next)
        slow.next = None
        p1, p2 = head, second
        ok = True
        while p2:
            if p1.val != p2.val:
                ok = False
                break
            p1 = p1.next
            p2 = p2.next
        slow.next = self._reverse(second)
        return ok

    def _reverse(self, head):
        prev = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        return prev
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

满足 O(1) 空间要求。快指针走两步、慢指针走一步，奇偶长度均适用。

## 解法二：数组

### 思路

值复制到数组，双指针从两端向中间比较。

### 代码

```python
from typing import Optional, List


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        vals: List[int] = []
        while head:
            vals.append(head.val)
            head = head.next
        left, right = 0, len(vals) - 1
        while left < right:
            if vals[left] != vals[right]:
                return False
            left += 1
            right -= 1
        return True
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

最简单，但不满足 O(1) 空间 follow-up。面试可先写此再优化。

## 解法三：递归

### 思路

递归到末尾，回溯时与前指针比较。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        self.front = head

        def check(node):
            if not node:
                return True
            if not check(node.next):
                return False
            if self.front.val != node.val:
                return False
            self.front = self.front.next
            return True

        return check(head)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) 递归栈 |

### 讲解

利用递归栈代替显式反转，空间 O(n)。

| 递归 | O(n) | O(n) | 技巧写法 |

## 常见错误

1. **快慢指针终止条件错**：应用 `fast.next and fast.next.next` 避免偶数长度越界。
2. **忘记断开前半与后半**：`slow.next = None`。
3. **与「回文数」混淆**：234 题是整数，本题是链表。
