# 删除链表的倒数第 N 个结点（Remove Nth Node From End of List）

- 题目链接：https://leetcode.cn/problems/remove-nth-node-from-end-of-list/
- 难度：中等
- 标签：链表、双指针

## 题目描述

给你一个链表，删除链表的**倒数第 n 个**结点，返回链表头。要求一趟扫描。

### 示例

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

输入：head = [1], n = 1
输出：[]

输入：head = [1,2], n = 1
输出：[1]
```

### 约束

- 链表中节点数 `k` 范围 `[1, 30]`
- `1 <= Node.val <= 100`
- `1 <= n <= k`

---

## 解法一：快慢指针 + 哑节点（推荐）

### 思路

快指针先走 n+1 步（配合哑节点），然后快慢同步走，慢指针停在待删节点**前驱**，删除 `slow.next`。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = slow = dummy
        for _ in range(n + 1):
            fast = fast.next
        while fast:
            fast = fast.next
            slow = slow.next
        slow.next = slow.next.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(L) |
| 空间 | O(1) |

### 讲解

走 n+1 步是为定位前驱。哑节点统一处理删头节点的情况。

## 解法二：两次遍历

### 思路

第一次数长度 L，第二次走到 L-n 的前驱删除。

### 代码

```python
from typing import Optional


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        length = 0
        cur = head
        while cur:
            length += 1
            cur = cur.next
        cur = dummy
        for _ in range(length - n):
            cur = cur.next
        cur.next = cur.next.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(L) |
| 空间 | O(1) |

### 讲解

两次遍历仍 O(L)，但不如一次扫描优雅。题目要求一趟时应用解法一。

## 解法三：栈

### 思路

节点入栈，弹出 n 个后栈顶为待删前驱（需特殊处理删头）。

### 代码

```python
from typing import Optional, List


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        stack: List[ListNode] = []
        cur = dummy
        while cur:
            stack.append(cur)
            cur = cur.next
        prev = stack[-(n + 1)]
        prev.next = prev.next.next
        return dummy.next
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(L) |
| 空间 | O(L) |

### 讲解

O(L) 额外空间，不如双指针优。

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 快慢指针 | O(L) | O(1) | 面试首选 |
| 两次遍历 | O(L) | O(1) | 直观写法 |
| 栈 | O(L) | O(L) | 辅助理解 |

## 常见错误

1. **快指针只走 n 步**：应走 n+1 步定位前驱。
2. **忘记哑节点**：删头节点 `[1], n=1` 会错。
3. **与「链表中倒数第 k 个节点」混淆**：剑指 Offer 22 只找不删。
