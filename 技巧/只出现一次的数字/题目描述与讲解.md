# 只出现一次的数字（Single Number）

- 题目链接：https://leetcode.cn/problems/single-number/
- 难度：简单
- 标签：位运算、数组

## 题目描述

给定一个**非空**整数数组 `nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法，且只使用常数额外空间。

### 示例

```
输入：nums = [2,2,1]
输出：1

输入：nums = [4,1,2,1,2]
输出：4

输入：nums = [1]
输出：1
```

### 约束

- `1 <= nums.length <= 3 × 10^4`
- `-3 × 10^4 <= nums[i] <= 3 × 10^4`
- 除了某个元素仅出现一次外，其余每个元素都恰出现两次

---

## 解法一：哈希表计数

### 思路

1. 用字典统计每个数出现次数。
2. 遍历字典，返回次数为 1 的键。

### 代码

```python
from typing import List
from collections import Counter


class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        for k, v in Counter(nums).items():
            if v == 1:
                return k
        return -1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

最直观，但额外空间 O(n)，不满足「常数空间」要求。适合先讲暴力再优化到位运算。

---

## 解法二：异或（推荐，与 solution.py 一致）

### 思路

1. 异或性质：`a ^ a = 0`，`a ^ 0 = a`，满足交换律、结合律。
2. 全部数异或后，成对的抵消，剩下唯一出现一次的数。

### 代码

```python
from typing import List


class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        ans = 0
        for x in nums:
            ans ^= x
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与 `solution.py` 一致。位运算经典题，面试必须秒答。注意负数在 Python 中同样适用异或。

---

## 解法三：排序后相邻比较

### 思路

排序后，落单的元素所在位置与前/后邻居都不同（或位于首尾）。

### 代码

```python
from typing import List


class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        nums.sort()
        n = len(nums)
        for i in range(0, n, 2):
            if i + 1 == n or nums[i] != nums[i + 1]:
                return nums[i]
        return -1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n log n) |
| 空间 | O(1) 或 O(log n) 排序栈 |

### 讲解

不满足线性时间，仅作对比。说明异或法的必要性。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 哈希计数 | O(n) | O(n) | 快速实现 |
| 异或 | O(n) | O(1) | 面试唯一正解 |
| 排序 | O(n log n) | O(1) | 理解用，超时 |

---

## 常见错误

1. **用加法/乘法**：无法区分成对抵消，必须用异或。
2. **与 136 变体混淆**：137 题一个数出现三次，需位计数；260 题两个只出现一次。
3. **空数组**：题目保证非空。
