# 颜色分类（Sort Colors）

- 题目链接：https://leetcode.cn/problems/sort-colors/
- 难度：中等
- 标签：数组、双指针、分治、排序

## 题目描述

给定一个包含红色、白色和蓝色、共 `n` 个元素的数组 `nums`，**原地**对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 0、1 和 2 分别表示红色、白色和蓝色。

你必须不使用库函数的一键 sort 来解决这个问题，且仅使用常数空间。

### 示例

```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]

输入：nums = [2,0,1]
输出：[0,1,2]
```

### 约束

- `n == nums.length`
- `1 <= n <= 300`
- `nums[i]` 为 0、1 或 2

---

## 解法一：计数排序

### 思路

1. 统计 0、1、2 的个数。
2. 按顺序填回数组：先全 0，再全 1，再全 2。

### 代码

```python
from typing import List


class Solution:
    def sortColors(self, nums: List[int]) -> None:
        cnt = [0, 0, 0]
        for x in nums:
            cnt[x] += 1
        i = 0
        for c in range(3):
            for _ in range(cnt[c]):
                nums[i] = c
                i += 1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

两遍扫描，满足常数空间。面试通常要求一趟扫描的荷兰国旗。

---

## 解法二：荷兰国旗 — 三路划分（推荐，与 solution.py 一致）

### 思路

1. 三指针：`lo` 指向下一个 0 的位置，`mid` 当前扫描，`hi` 指向下一个 2 的位置。
2. `nums[mid]==0`：与 `lo` 交换，双指针右移。
3. `nums[mid]==1`：`mid` 右移。
4. `nums[mid]==2`：与 `hi` 交换，`hi` 左移（**mid 不动**，换过来的可能是 0/2）。

### 代码

```python
from typing import List


class Solution:
    def sortColors(self, nums: List[int]) -> None:
        lo, mid, hi = 0, 0, len(nums) - 1
        while mid <= hi:
            if nums[mid] == 0:
                nums[lo], nums[mid] = nums[mid], nums[lo]
                lo += 1
                mid += 1
            elif nums[mid] == 1:
                mid += 1
            else:
                nums[mid], nums[hi] = nums[hi], nums[mid]
                hi -= 1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与 `solution.py` 一致。Invariant：`[0..lo-1]` 为 0，`[lo..mid-1]` 为 1，`[hi+1..n-1]` 为 2。交换 2 时 mid 不增是关键。

## 解法三：双指针两次扫描

### 思路

1. 第一趟：把 0 换到前面，维护边界 `p0`。
2. 第二趟：在剩余部分把 1 换到前面，维护 `p1`。
3. 类似快排 partition 两次。

### 代码

```python
from typing import List


class Solution:
    def sortColors(self, nums: List[int]) -> None:
        p0 = p1 = 0
        for x in nums:
            if x == 1:
                nums[p1], nums[p0] = nums[p0], nums[p1]
                p1 += 1
            elif x == 2:
                nums[p1], nums[p0] = nums[p0], nums[p1]
                p1 += 1
                nums[p0], nums[p1 - 1] = nums[p1 - 1], nums[p0]
                p0 += 1
                p1 += 1
            else:
                p0 += 1
                p1 += 1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

逻辑较绕，不如荷兰国旗清晰。了解即可，面试写解法二。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 计数 | O(n) | O(1) | 两遍扫描可接受 |
| 荷兰国旗 | O(n) | O(1) | 面试首选，一趟 |
| 双指针两次 | O(n) | O(1) | 理解 partition |

---

## 常见错误

1. **换 2 时 mid++**：换过来的元素未检查，必须 mid 不动。
2. **循环条件**：`while mid <= hi`，不是 `<`。
3. **与快排混淆**：本题仅三值，专用算法更优。
