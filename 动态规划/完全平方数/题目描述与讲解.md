# 完全平方数（Perfect Squares）

- 题目链接：https://leetcode.cn/problems/perfect-squares/
- 难度：中等
- 标签：广度优先搜索、数学、动态规划

## 题目描述

给你一个整数 `n`，返回和为 `n` 的完全平方数的最少数量。

**完全平方数** 是一个整数，可以表示为某个整数的平方，例如 1、4、9、16 等。

### 示例

```
输入：n = 12
输出：3
解释：12 = 4 + 4 + 4

输入：n = 13
输出：2
解释：13 = 4 + 9
```

### 约束

- `1 <= n <= 10^4`

---

## 解法一：BFS 按层扩展

### 思路

1. 把问题看作：从 0 出发，每次加上一个完全平方数，最少几步到 n。
2. 用 BFS，第一层加 1²，第二层再加平方数……最先到达 n 的层数即答案。
3. 预先生成所有 ≤ n 的完全平方数列表。

### 代码

```python
from collections import deque


class Solution:
    def numSquares(self, n: int) -> int:
        squares = [i * i for i in range(1, int(n ** 0.5) + 1)]
        queue = deque([(0, 0)])  # (当前和, 步数)
        visited = {0}
        while queue:
            s, step = queue.popleft()
            for sq in squares:
                ns = s + sq
                if ns == n:
                    return step + 1
                if ns < n and ns not in visited:
                    visited.add(ns)
                    queue.append((ns, step + 1))
        return 0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × √n) 量级 |
| 空间 | O(n) |

### 讲解

BFS 保证第一次到达 n 时步数最少。适合理解「最少步数」类问题，但 DP 通常更简洁。

## 解法二：完全背包 DP（推荐，与 solution.py 一致）

### 思路

1. `dp[i]`：和为 `i` 所需最少完全平方数个数。
2. 对每个 `i`，枚举完全平方数 `sq ≤ i`：`dp[i] = min(dp[i], dp[i-sq] + 1)`。
3. 初始化 `dp[0]=0`，其余为无穷大。

### 代码

```python
class Solution:
    def numSquares(self, n: int) -> int:
        dp = [0] + [float("inf")] * n
        squares = [i * i for i in range(1, int(n ** 0.5) + 1)]
        for i in range(1, n + 1):
            for sq in squares:
                if sq > i:
                    break
                dp[i] = min(dp[i], dp[i - sq] + 1)
        return dp[n]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × √n) |
| 空间 | O(n) |

### 讲解

与 `solution.py` 一致。本质是「无限物品背包」：物品是完全平方数，价值为 1，求凑满容量 n 的最少物品数。

## 解法三：数学 — 四平方和定理

### 思路

拉格朗日四平方和定理：任意正整数最多可表示为 4 个完全平方数之和。可依次判断能否用 1、2、3 个完全平方数表示，否则答案为 4。

### 代码

```python
class Solution:
    def numSquares(self, n: int) -> int:
        def is_square(x: int) -> bool:
            r = int(x ** 0.5)
            return r * r == x

        # 1 个平方数
        if is_square(n):
            return 1
        # 2 个平方数
        i = 1
        while i * i <= n:
            if is_square(n - i * i):
                return 2
            i += 1
        # 4 的倍数中的 4 次方因子
        while n % 4 == 0:
            n //= 4
        return 3 if n % 8 == 7 else 4
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(√n) |
| 空间 | O(1) |

### 讲解

理论最优但实现复杂，面试中 DP 更稳妥。了解定理可展示数学功底。

## 常见错误

1. **初始化**：`dp[0]=0`，不是 1 或 inf。
2. **平方数枚举**：只需到 `√n`，且 `sq > i` 时可 break。
3. **与「分割整数」混淆**：本题是完全平方数，不是任意正整数拆分。
