# 爬楼梯（Climbing Stairs）

- 题目链接：https://leetcode.cn/problems/climbing-stairs/
- 难度：简单
- 标签：记忆化搜索、数学、动态规划

## 题目描述

假设你正在爬楼梯。需要 `n` 阶才能到达楼顶。

每次你可以爬 **1** 或 **2** 个台阶。你有多少种不同的方法可以爬到楼顶？

### 示例

```
输入：n = 2
输出：2
解释：有两种方法：1+1 或 2

输入：n = 3
输出：3
解释：有三种方法：1+1+1、1+2、2+1
```

### 约束

- `1 <= n <= 45`

---

## 解法一：递归 + 记忆化（理解 DP 来源）

### 思路

1. 到达第 `n` 阶，最后一步要么从 `n-1` 来，要么从 `n-2` 来。
2. 因此 `f(n) = f(n-1) + f(n-2)`，边界 `f(1)=1, f(2)=2`。
3. 纯递归会重复计算，用字典或数组缓存已算过的值。

### 代码

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        memo = {1: 1, 2: 2}

        def dfs(k: int) -> int:
            if k in memo:
                return memo[k]
            memo[k] = dfs(k - 1) + dfs(k - 2)
            return memo[k]

        return dfs(n)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，每个状态只算一次 |
| 空间 | O(n)，递归栈 + 缓存 |

### 讲解

这是斐波那契数列的经典应用。记忆化把指数级递归变成线性 DP，适合第一次接触「状态转移」时理解。

## 解法二：自底向上 DP 数组

### 思路

1. 定义 `dp[i]` 为爬到第 `i` 阶的方法数。
2. 初始化 `dp[1]=1, dp[2]=2`。
3. 从 3 到 n 递推：`dp[i] = dp[i-1] + dp[i-2]`。

### 代码

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        dp = [0] * (n + 1)
        dp[1], dp[2] = 1, 2
        for i in range(3, n + 1):
            dp[i] = dp[i - 1] + dp[i - 2]
        return dp[n]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

自底向上避免了递归开销，逻辑清晰。面试中可先写此版本，再优化空间。

## 解法三：滚动变量（推荐，与 solution.py 一致）

### 思路

`dp[i]` 只依赖前两项，用两个变量 `a, b` 滚动更新即可，无需整表。

### 代码

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b
        return b
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

最优写法，与 `solution.py` 一致。注意 `n=1` 返回 1、`n=2` 返回 2，不要误用斐波那契下标偏移。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 记忆化递归 | O(n) | O(n) | 理解状态转移 |
| DP 数组 | O(n) | O(n) | 需要路径/方案数细节时 |
| 滚动变量 | O(n) | O(1) | 面试提交首选 |

## 常见错误

1. **边界处理**：`n=1` 应返回 1，不是 0。
2. **与「最小代价爬楼梯」混淆**：70 题求最小代价，本题求方案数。
3. **矩阵快速幂误用**：n 最大 45，线性 DP 足够，不必上快速幂。
