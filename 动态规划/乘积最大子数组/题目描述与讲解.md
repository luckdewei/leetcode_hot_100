# 乘积最大子数组（Maximum Product Subarray）

- 题目链接：https://leetcode.cn/problems/maximum-product-subarray/
- 难度：中等
- 标签：数组、动态规划

## 题目描述

给你一个整数数组 `nums`，请你找出数组中乘积最大的非空**连续**子数组（子数组最少包含一个数），并返回对应的乘积。

测试用例保证答案是一个 32 位整数。

### 示例

```
输入：nums = [2,3,-2,4]
输出：6
解释：子数组 [2,3] 乘积最大

输入：nums = [-2,0,-1]
输出：0
解释：结果不能是 2，因为 [-2,-1] 不是连续子数组
```

### 约束

- `1 <= nums.length <= 2 × 10^4`
- `-10 <= nums[i] <= 10`
- `nums` 的任何子数组乘积都在 32 位整数范围内

---

## 解法一：DP 维护最大/最小乘积（推荐，与 solution.py 一致）

### 思路

1. 负数会使最大变最小，因此同时维护 `max_dp` 和 `min_dp`（以当前位置结尾）。
2. 遇到负数时，交换 max 与 min（因为乘负数会翻转大小关系）。
3. 更新：`max_dp = max(x, max_dp * x)`，`min_dp = min(x, min_dp * x)`。
4. 全局答案取所有 `max_dp` 的最大值。

### 代码

```python
from typing import List


class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        ans = nums[0]
        max_dp = min_dp = 1
        for x in nums:
            if x < 0:
                max_dp, min_dp = min_dp, max_dp
            max_dp = max(x, max_dp * x)
            min_dp = min(x, min_dp * x)
            ans = max(ans, max_dp)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与 `solution.py` 一致。核心洞察：最小乘积遇到负数可能变成最大。初始 `max_dp=min_dp=1` 表示「空后缀」的乘积为 1，与 `x` 比较可处理单元素子数组。

## 解法二：显式三分支 DP 数组

### 思路

不显式交换，每次用上一轮的 max/min 计算三个候选：只取 x、接 max、接 min。

### 代码

```python
from typing import List


class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        n = len(nums)
        max_dp = [0] * n
        min_dp = [0] * n
        max_dp[0] = min_dp[0] = nums[0]
        ans = nums[0]
        for i in range(1, n):
            x = nums[i]
            candidates = [x, max_dp[i - 1] * x, min_dp[i - 1] * x]
            max_dp[i] = max(candidates)
            min_dp[i] = min(candidates)
            ans = max(ans, max_dp[i])
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

逻辑更清晰，适合第一次推导。可进一步压成 O(1) 空间即为解法一。

## 解法三：记录负数个数的暴力优化（理解用）

### 思路

扫描时记录当前段的最大/最小前缀积；遇 0 则重置。等价于 DP，用变量 `cur_max, cur_min` 滚动。

### 代码

```python
from typing import List


class Solution:
    def maxProduct(self, nums: List[int]) -> int:
        ans = cur_max = cur_min = nums[0]
        for i in range(1, len(nums)):
            x = nums[i]
            if x < 0:
                cur_max, cur_min = cur_min, cur_max
            cur_max = max(x, cur_max * x)
            cur_min = min(x, cur_min * x)
            ans = max(ans, cur_max)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与解法一同构，变量名不同。不推荐 O(n²) 枚举所有子数组，必超时。

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 滚动 max/min | O(n) | O(1) | 面试首选 |
| DP 数组 | O(n) | O(n) | 调试转移 |
| 前缀积扫描 | O(n) | O(1) | 同解法一 |

## 常见错误

1. **只维护最大值**：忽略 min_dp，负负得正会漏解。
2. **与最大子数组和混淆**：53 题是加法，本题是乘法，且需 min 状态。
3. **初始值**：应用 `nums[0]` 初始化 ans，不是 0（全负数时答案为负）。
