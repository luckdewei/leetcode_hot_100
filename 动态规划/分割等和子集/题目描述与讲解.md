# 分割等和子集（Partition Equal Subset Sum）

- 题目链接：https://leetcode.cn/problems/partition-equal-subset-sum/
- 难度：中等
- 标签：数组、动态规划

## 题目描述

给你一个**只包含正整数**的非空数组 `nums`。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

### 示例

```
输入：nums = [1,5,11,5]
输出：true
解释：数组可以分割成 [1,5,5] 和 [11]

输入：nums = [1,2,3,5]
输出：false
解释：数组不能分割成两个元素和相等的子集
```

### 约束

- `1 <= nums.length <= 200`
- `1 <= nums[i] <= 100`

---

## 解法一：递归 + 记忆化（0-1 背包）

### 思路

1. 若总和为奇数，直接 false。
2. 目标：是否存在子集和为 `sum/2`。
3. `dfs(i, rem)`：前 i 个数能否凑出 rem；选或不选第 i 个数。

### 代码

```python
from typing import List
from functools import lru_cache


class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2:
            return False
        target = total // 2

        @lru_cache(maxsize=None)
        def dfs(i: int, rem: int) -> bool:
            if rem == 0:
                return True
            if i < 0 or rem < 0:
                return False
            if dfs(i - 1, rem):
                return True
            return dfs(i - 1, rem - nums[i])

        return dfs(len(nums) - 1, target)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × target) |
| 空间 | O(n × target) |

### 讲解

转化为「子集和是否存在」，是 0-1 背包的判定版。记忆化清晰但 Python 递归深度有限，大数据用迭代更好。

## 解法二：0-1 背包 DP 数组（推荐，与 solution.py 一致）

### 思路

1. `dp[j]`：能否凑出和为 j（布尔）。
2. 对每个数 `num`，**倒序**遍历 j 从 target 到 num：`dp[j] |= dp[j-num]`。
3. 返回 `dp[target]`。

### 代码

```python
from typing import List


class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2:
            return False
        target = total // 2
        dp = [False] * (target + 1)
        dp[0] = True
        for num in nums:
            for j in range(target, num - 1, -1):
                dp[j] = dp[j] or dp[j - num]
        return dp[target]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × target) |
| 空间 | O(target) |

### 讲解

与 `solution.py` 一致。倒序保证每个数只用一次。`dp[0]=True` 表示空集和为 0。

## 解法三：位运算优化（进阶）

### 思路

用整数 bitmask 表示可达的和：初始 `bits=1`，每个 num 做 `bits |= bits << num`。

### 代码

```python
from typing import List


class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        total = sum(nums)
        if total % 2:
            return False
        target = total // 2
        bits = 1
        for num in nums:
            bits |= bits << num
        return (bits >> target) & 1 == 1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × target / word_size) 近似 |
| 空间 | O(1) |

### 讲解

target 最大约 100×200/2=10000，Python 大整数位运算仍可行。面试中解法二更通用。

|------|------|------|----------|
| 记忆化 | O(nT) | O(nT) | 理解背包 |
| 0-1 背包 DP | O(nT) | O(T) | 面试首选 |
| 位运算 | O(nT/w) | O(1) | 常数优化 |

*T=target=sum/2*

## 常见错误

1. **忘记奇数和判否**：总和为奇数不可能平分。
2. **正序遍历 j**：0-1 背包必须倒序，否则会重复使用同一元素。
3. **与 416 变体混淆**：本题只要判断存在，不要求输出具体划分。
