# 打家劫舍（House Robber）

- 题目链接：https://leetcode.cn/problems/house-robber/
- 难度：中等
- 标签：数组、动态规划

## 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定现金，**相邻的房屋装有相互连通的防盗系统**，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个非负整数数组 `nums`，表示每间房屋中的金额。计算你**在不触动报警装置**的情况下，一夜之内能够偷窃到的最高金额。

### 示例

```
输入：nums = [1,2,3,1]
输出：4
解释：偷 1 号（1）和 3 号（3），总计 4

输入：nums = [2,7,9,3,1]
输出：12
解释：偷 2、9、1，总计 12
```

### 约束

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

---

## 解法一：递归 + 记忆化

### 思路

1. 定义 `dfs(i)`：考虑前 `i` 间房屋（下标 0..i）能偷的最大金额。
2. 第 `i` 间要么偷（则 `i-1` 不能偷）：`nums[i] + dfs(i-2)`；要么不偷：`dfs(i-1)`。
3. 转移：`dfs(i) = max(dfs(i-1), dfs(i-2) + nums[i])`。

### 代码

```python
from typing import List


class Solution:
    def rob(self, nums: List[int]) -> int:
        memo = {}

        def dfs(i: int) -> int:
            if i < 0:
                return 0
            if i in memo:
                return memo[i]
            memo[i] = max(dfs(i - 1), dfs(i - 2) + nums[i])
            return memo[i]

        return dfs(len(nums) - 1)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

经典「选或不选」DP。记忆化帮助理解：当前决策只依赖「上一间偷没偷」的状态。

## 解法二：DP 数组

### 思路

1. `dp[i]` 表示考虑前 `i+1` 间房的最大收益。
2. `dp[i] = max(dp[i-1], dp[i-2] + nums[i])`。
3. 初始 `dp[0] = nums[0]`，`dp[1] = max(nums[0], nums[1])`。

### 代码

```python
from typing import List


class Solution:
    def rob(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0]
        dp = [0] * n
        dp[0], dp[1] = nums[0], max(nums[0], nums[1])
        for i in range(2, n):
            dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
        return dp[-1]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

表格形式便于调试。注意 `dp[i-1]` 表示「不偷第 i 间」，`dp[i-2]+nums[i]` 表示「偷第 i 间」。

## 解法三：滚动变量（推荐，与 solution.py 一致）

### 思路

只需前两个状态：`prev2`（i-2 最优）、`prev1`（i-1 最优），逐屋滚动更新。

### 代码

```python
from typing import List


class Solution:
    def rob(self, nums: List[int]) -> int:
        prev2 = prev1 = 0
        for x in nums:
            prev2, prev1 = prev1, max(prev1, prev2 + x)
        return prev1
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

与 `solution.py` 一致，面试最优写法。`prev2, prev1 = prev1, max(...)` 一行完成状态转移。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 记忆化 | O(n) | O(n) | 理解递归结构 |
| DP 数组 | O(n) | O(n) | 需要路径信息时 |
| 滚动变量 | O(n) | O(1) | 提交首选 |

## 常见错误

1. **相邻约束**：不能连续偷相邻两间，不是「不能偷同一间两次」。
2. **空数组**：题目保证 length≥1。
3. **与 213 题混淆**：打家劫舍 II 是环形，首尾相邻需特殊处理。
