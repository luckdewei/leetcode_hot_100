# 杨辉三角（Pascal's Triangle）

- 题目链接：https://leetcode.cn/problems/pascals-triangle/
- 难度：简单
- 标签：数组、动态规划

## 题目描述

给定一个非负整数 `numRows`，生成「杨辉三角」的前 `numRows` 行。

在杨辉三角中，每个数等于它上方两数之和：

```
     1
    1 1
   1 2 1
  1 3 3 1
 1 4 6 4 1
```

### 示例

```
输入：numRows = 5
输出：[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]

输入：numRows = 1
输出：[[1]]
```

### 约束

- `1 <= numRows <= 30`

---

## 解法一：逐行模拟（推荐，与 solution.py 一致）

### 思路

1. 第 `i` 行（0-indexed）长度为 `i+1`，首尾均为 1。
2. 中间元素：`row[j] = prev[j-1] + prev[j]`，其中 `prev` 是上一行。
3. 循环 `numRows` 次，每行构造后追加到答案。

### 代码

```python
from typing import List


class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        ans = []
        for i in range(numRows):
            row = [1] * (i + 1)
            for j in range(1, i):
                row[j] = ans[i - 1][j - 1] + ans[i - 1][j]
            ans.append(row)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(numRows²)，共约 numRows²/2 个元素 |
| 空间 | O(numRows²)，不含输出则 O(numRows) 滚动一行 |

### 讲解

最直观的写法，与 `solution.py` 一致。内层 `j` 从 1 到 `i-1`，避免改首尾。

## 解法二：组合数公式

### 思路

第 `i` 行第 `j` 列（0-indexed）等于组合数 C(i, j) = i! / (j! × (i-j)!)。

逐元素用公式计算，无需依赖上一行。

### 代码

```python
from typing import List


class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        ans = []
        for i in range(numRows):
            row = []
            for j in range(i + 1):
                if j == 0 or j == i:
                    row.append(1)
                else:
                    row.append(ans[i - 1][j - 1] + ans[i - 1][j])
            ans.append(row)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(numRows²) |
| 空间 | O(numRows²) |

### 讲解

与解法一等价，只是写法上显式处理首尾。也可用递推 C(i,j)=C(i,j-1)×(i-j+1)/j 避免阶乘溢出。

## 解法三：滚动一行（空间优化）

### 思路

只保留上一行，当前行在原位更新或新建，适合「杨辉三角 II」只求第 n 行。

### 代码

```python
from typing import List


class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        ans = [[1]]
        for i in range(1, numRows):
            prev = ans[-1]
            row = [1]
            for j in range(1, i):
                row.append(prev[j - 1] + prev[j])
            row.append(1)
            ans.append(row)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(numRows²) |
| 空间 | O(numRows²) 输出；中间 O(numRows) |

### 讲解

首尾单独 append，逻辑更对称。Hot 100 中 119 题「杨辉三角 II」只需返回最后一行，可在此基础上优化。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 逐行模拟 | O(n²) | O(n²) | 面试首选 |
| 组合数 | O(n²) | O(n²) | 数学背景强时 |
| 滚动一行 | O(n²) | O(n) 中间 | 只求单行时 |

## 常见错误

1. **内层循环边界**：`j in range(1, i)`，不是 `range(i)`，否则会覆盖首尾。
2. **空输入**：题目保证 numRows≥1，无需特判 0。
3. **整数溢出**：Python 无此问题，其他语言需注意大组合数。
