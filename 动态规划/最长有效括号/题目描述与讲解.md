# 最长有效括号（Longest Valid Parentheses）

- 题目链接：https://leetcode.cn/problems/longest-valid-parentheses/
- 难度：困难
- 标签：栈、字符串、动态规划

## 题目描述

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

### 示例

```
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"

输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"

输入：s = ""
输出：0
```

### 约束

- `0 <= s.length <= 3 × 10^4`
- `s[i]` 为 `'('` 或 `')'`

---

## 解法一：栈（推荐，与 solution.py 一致）

### 思路

1. 栈底压入 `-1` 作为「有效段起点的前一位」哨兵。
2. 遇 `'('` 压入下标；遇 `')'` 弹出。
3. 若栈空，说明 `')'` 无法匹配，将其下标压栈作新起点哨兵。
4. 若栈非空，当前有效长度 = `i - stack[-1]`，更新答案。

### 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        stack = [-1]
        ans = 0
        for i, ch in enumerate(s):
            if ch == "(":
                stack.append(i)
            else:
                stack.pop()
                if not stack:
                    stack.append(i)
                else:
                    ans = max(ans, i - stack[-1])
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

与 `solution.py` 一致。栈存下标而非字符，便于算长度。哨兵 `-1` 处理从 0 开始的有效段，如 `"()"` 长度为 `1-(-1)=2`。

## 解法二：双向扫描

### 思路

1. 从左到右扫：维护 `left, right` 计数；`right>left` 时重置；相等时更新 ans。
2. 从右到左再扫一遍（处理 `"(()"` 类左括号多余的情况）。
3. 两次扫描取最大值。

### 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        def scan(s: str, lch: str, rch: str) -> int:
            left = right = ans = 0
            for ch in s:
                if ch == lch:
                    left += 1
                else:
                    right += 1
                if left == right:
                    ans = max(ans, 2 * right)
                elif right > left:
                    left = right = 0
            return ans

        return max(scan(s, "(", ")"), scan(s[::-1], ")", "("))
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1) |

### 讲解

O(1) 空间技巧题。反向扫描时把 `(` 和 `)` 角色互换，弥补单次左扫的盲区。

## 解法三：DP

### 思路

`dp[i]`：以 `s[i]` **结尾**的最长有效括号长度。若 `s[i]==')'` 且 `s[i-1]=='('`，则 `dp[i]=dp[i-2]+2`；若 `s[i-1]==')'` 且前面有有效段，可拼接。

### 代码

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        dp = [0] * n
        ans = 0
        for i in range(1, n):
            if s[i] != ")":
                continue
            if s[i - 1] == "(":
                dp[i] = (dp[i - 2] if i >= 2 else 0) + 2
            elif i - dp[i - 1] - 1 >= 0 and s[i - dp[i - 1] - 1] == "(":
                dp[i] = dp[i - 1] + 2
                if i - dp[i - 1] - 2 >= 0:
                    dp[i] += dp[i - dp[i - 1] - 2]
            ans = max(ans, dp[i])
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

转移较绕，面试栈法更稳。DP 适合已熟悉「以 i 结尾」框架的同学。

## 常见错误

1. **栈空未重置哨兵**：`')'` 无法匹配时应 `stack.append(i)`，不是不管。
2. **有效子串必须连续**：子序列不行，如 `"( )"` 中间有空格在本题无，但 `")("` 不算。
3. **与有效括号**（20 题）混淆：20 题判断整体是否有效，本题求最长连续有效段。
