# leetcode_hot_100

力扣 Hot 100 完整题解，共 **100** 题，按官方题单类别分组。

## 目录结构

```
类别名/
  └── 题目名/
        ├── solution.py          # Python 最优解 + 本地测试
        └── 题目描述与讲解.md     # 题目描述、多种解法、详细讲解
```

## 使用方式

1. 按类别进入对应文件夹，阅读 `题目描述与讲解.md` 理解思路
2. 对照 `solution.py` 中的实现与本地测试用例
3. 建议同类别连续刷题，更容易形成解题套路

## 本地测试

```bash
python3 哈希/两数之和/solution.py

# 批量运行全部测试
find . -name "solution.py" -exec python3 {} \;
```
