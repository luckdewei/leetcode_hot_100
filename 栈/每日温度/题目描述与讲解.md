# 每日温度（Daily Temperatures）

- 题目链接：https://leetcode.cn/problems/daily-temperatures/
- 难度：中等
- 标签：栈、数组、单调栈

## 题目描述

给定一个整数数组 `temperatures`，表示每天的温度，返回一个数组 `answer`，其中 `answer[i]` 是指在第 `i` 天之后，要等多少天才能遇到更高的温度。若不存在则 `answer[i] = 0`。

### 示例

```
输入：temperatures = [73,74,75,71,69,72,76,73]
输出：[1,1,4,2,1,1,0,0]

输入：temperatures = [30,40,50,60]
输出：[1,1,1,0]

输入：temperatures = [30,60,90]
输出：[1,1,0]
```

### 约束

- `1 <= temperatures.length <= 10^5`
- `30 <= temperatures[i] <= 100`

---

## 解法一：单调递减栈（推荐）

### 思路

栈存下标，保持栈内温度单调递减。当前温度高于栈顶下标对应温度时，栈顶元素的答案 = 当前下标 - 栈顶下标，弹出栈顶。

### 代码

```python
from typing import List


class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        n = len(temperatures)
        ans = [0] * n
        stack = []
        for i, t in enumerate(temperatures):
            while stack and temperatures[stack[-1]] < t:
                j = stack.pop()
                ans[j] = i - j
            stack.append(i)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

每个下标最多入栈出栈一次。栈内维护「尚未找到更高温度的天数」的下标。

## 解法二：逆序遍历 + 跳表

### 思路

从右向左，对每个 i，若 `T[i+1] <= T[i]` 则跳过已算过的步数，直到找到更高温度或到末尾。

### 代码

```python
from typing import List


class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        n = len(temperatures)
        ans = [0] * n
        for i in range(n - 2, -1, -1):
            j = i + 1
            while j < n and temperatures[j] <= temperatures[i]:
                if ans[j] == 0:
                    j = n
                    break
                j += ans[j]
            if j < n:
                ans[i] = j - i
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) 均摊 |
| 空间 | O(1) 额外 |

### 讲解

利用已计算的 `ans[j]` 跳跃，均摊 O(n)。不如单调栈直观，但空间更优。

## 解法三：暴力（理解用）

### 思路

对每个 i，向后找第一个更高温度。

### 代码

```python
from typing import List


class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        n = len(temperatures)
        ans = [0] * n
        for i in range(n):
            for j in range(i + 1, n):
                if temperatures[j] > temperatures[i]:
                    ans[i] = j - i
                    break
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(1) |

### 讲解

O(n²) 会超时，帮助理解「下一个更大元素」问题。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 单调栈 | O(n) | O(n) | 面试首选 |
| 逆序跳表 | O(n) | O(1) | 空间优化 |
| 暴力 | O(n²) | O(1) | 理解用 |

## 常见错误

1. **栈存温度而非下标**：需下标算天数差。
2. **比较用 `<=` 而非 `<`**：相等温度不算更高。
3. **与「柱状图最大矩形」混淆**：矩形用递增栈，温度用递减栈。
