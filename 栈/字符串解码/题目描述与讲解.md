# 字符串解码（Decode String）

- 题目链接：https://leetcode.cn/problems/decode-string/
- 难度：中等
- 标签：栈、字符串、递归

## 题目描述

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则：`k[encoded_string]` 表示 `encoded_string` 重复 `k` 次。

- `1 <= k <= 300`
- 括号可嵌套

### 示例

```
输入：s = "3[a]2[bc]"
输出："aaabcbc"

输入：s = "3[a2[c]]"
输出："accaccacc"

输入：s = "2[abc]3[cd]ef"
输出："abcabccdcdcdef"
```

### 约束

- `1 <= s.length <= 30`
- `s` 由小写英文字母、数字和 `[]` 组成
- `s` 保证有效

---

## 解法一：栈（推荐）

### 思路

遍历字符：
- **数字**：累乘构建 `cur_num`（可能是多位数）
- **`[`**：把当前 `(cur_str, cur_num)` 压栈，重置
- **`]`**：弹栈得到 `(prev_str, repeat)`，`cur_str = prev_str + cur_str * repeat`
- **字母**：追加到 `cur_str`

### 代码

```python
class Solution:
    def decodeString(self, s: str) -> str:
        stack = []
        cur_num = 0
        cur_str = ""
        for ch in s:
            if ch.isdigit():
                cur_num = cur_num * 10 + int(ch)
            elif ch == "[":
                stack.append((cur_str, cur_num))
                cur_str, cur_num = "", 0
            elif ch == "]":
                prev_str, repeat = stack.pop()
                cur_str = prev_str + cur_str * repeat
            else:
                cur_str += ch
        return cur_str
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n × max_k)，输出长度可能很大 |
| 空间 | O(n) 栈深度 |

## 解法二：递归

遇到 `[` 递归 decode 子串直到 `]`，返回子串与结束位置。思路与栈等价，代码更函数式。

### 代码

```python
class Solution:
    def decodeString(self, s: str) -> str:
        def decode(i: int):
            res, num = "", 0
            while i < len(s):
                if s[i].isdigit():
                    num = num * 10 + int(s[i])
                elif s[i] == "[":
                    sub, i = decode(i + 1)
                    res += sub * num
                    num = 0
                elif s[i] == "]":
                    return res, i
                else:
                    res += s[i]
                i += 1
            return res, i

        return decode(0)[0]
```

## 解法三：从内向外逐层展开（不推荐）

反复找最内层 `[...]` 替换，实现繁琐。

### 代码

```python
import re


class Solution:
    def decodeString(self, s: str) -> str:
        while "[" in s:
            s = re.sub(r"(\d+)\[([^\[\]]*)\]", lambda m: m.group(2) * int(m.group(1)), s)
        return s
```

---

## 讲解

**多位数 k**：不能遇数字就立即入栈，需 `cur_num = cur_num * 10 + digit`。

**嵌套**：外层 `]` 展开时，内层结果已在 `cur_str` 中。

---

## 常见错误

1. **忘记重置 cur_num**：进入 `[` 后 num 应置 0。
2. **顺序错误**：应是 `prev_str + cur_str * repeat`，不是反过来。
3. **数字与 `[` 之间无分隔**：如 `100[a]` 需连续读数字。
