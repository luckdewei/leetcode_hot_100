# 有效的括号（Valid Parentheses）

- 题目链接：https://leetcode.cn/problems/valid-parentheses/
- 难度：简单
- 标签：栈、字符串

## 题目描述

给定一个只包括 `'('`、`')'`、`'{'`、`'}'`、`'['`、`']'` 的字符串 `s`，判断字符串是否有效。

有效需满足：左括号必须用相同类型右括号闭合；左括号必须以正确顺序闭合；每个右括号都有对应的左括号。

### 示例

```
输入：s = "()"
输出：true

输入：s = "()[]{}"
输出：true

输入：s = "(]"
输出：false

输入：s = "([)]"
输出：false
```

### 约束

- `1 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成

---

## 解法一：栈匹配（推荐）

### 思路

遇左括号入栈；遇右括号检查栈顶是否为对应左括号，是则弹出，否则无效。最终栈空则有效。

### 代码

```python
class Solution:
    def isValid(self, s: str) -> bool:
        pairs = {")": "(", "]": "[", "}": "{"}
        stack = []
        for ch in s:
            if ch in pairs:
                if not stack or stack[-1] != pairs[ch]:
                    return False
                stack.pop()
            else:
                stack.append(ch)
        return not stack
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

用哈希表存右括号到左括号的映射。栈空时遇右括号直接返回 False。

---

## 解法二：栈 + 成对判断

### 思路

遇左括号入栈；遇右括号时若栈空或栈顶不匹配则 False，否则弹出。

### 代码

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        match = {"(": ")", "[": "]", "{": "}"}
        for ch in s:
            if ch in match:
                stack.append(ch)
            else:
                if not stack:
                    return False
                top = stack.pop()
                if match[top] != ch:
                    return False
        return len(stack) == 0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

与解法一等价，分别用右查左和左查右。面试任选其一。

---

## 解法三：替换消去法（理解用）

### 思路

反复将 `"()"`、`"[]"`、`"{}"` 替换为空，最终串空则有效。

### 代码

```python
class Solution:
    def isValid(self, s: str) -> bool:
        while "()" in s or "[]" in s or "{}" in s:
            s = s.replace("()", "").replace("[]", "").replace("{}", "")
        return s == ""
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) 最坏 |
| 空间 | O(n) |

### 讲解

每次 replace 可能产生新串，效率低，不可用于大规模数据。

## 常见错误

1. **只检查栈空不检查匹配类型**：`"(]"` 栈不空但类型错。
2. **忘记最终栈须为空**：`"((("` 遍历结束栈非空。
3. **与「最长有效括号」混淆**：32 题求最长长度，本题判整体有效。
