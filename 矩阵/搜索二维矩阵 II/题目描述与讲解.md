# 搜索二维矩阵 II（Search a 2D Matrix II）

- 题目链接：https://leetcode.cn/problems/search-a-2d-matrix-ii/
- 难度：中等
- 标签：数组、二分查找、分治

## 题目描述

编写一个高效的算法来搜索 `m x n` 矩阵 `matrix` 中的目标值 `target`。该矩阵满足：每行从左到右升序，每列从上到下升序。

### 示例

```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true

输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 20
输出：false
```

### 约束

- `m == matrix.length`, `n == matrix[i].length`
- `1 <= n, m <= 300`
- `-10^9 <= matrix[i][j], target <= 10^9`

---

## 解法一：从右上角搜索（推荐）

### 思路

从右上角出发：当前值大于 target 则左移（排除当前列），小于 target 则下移（排除当前行）。

### 代码

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        if not matrix or not matrix[0]:
            return False
        row, col = 0, len(matrix[0]) - 1
        while row < len(matrix) and col >= 0:
            val = matrix[row][col]
            if val == target:
                return True
            if val > target:
                col -= 1
            else:
                row += 1
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m + n) |
| 空间 | O(1) |

### 讲解

每步排除一行或一列，最多 m+n 步。从左下角出发同理。

---

## 解法二：逐行二分

### 思路

每行有序，对每行二分查找 target。

### 代码

```python
from typing import List
import bisect


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        for row in matrix:
            idx = bisect.bisect_left(row, target)
            if idx < len(row) and row[idx] == target:
                return True
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m log n) |
| 空间 | O(1) |

### 讲解

行数少、列数多时可能更优。实现简单，依赖标准库二分。

---

## 解法三：暴力扫描（理解用）

### 思路

遍历矩阵每个元素比较。

### 代码

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        for row in matrix:
            for val in row:
                if val == target:
                    return True
        return False
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn) |
| 空间 | O(1) |

### 讲解

未利用行列有序性，会超时。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 右上角搜索 | O(m+n) | O(1) | 面试首选 |
| 逐行二分 | O(m log n) | O(1) | 行少列多 |
| 暴力 | O(mn) | O(1) | 理解用 |

## 常见错误

1. **与「搜索二维矩阵 I」混淆**：I 题整体有序可一次二分，II 题只能逐行或 Z 形搜索。
2. **从左下角/右上角搞反**：方向错则排除逻辑错。
3. **空矩阵未处理**：`matrix` 或 `matrix[0]` 为空。
