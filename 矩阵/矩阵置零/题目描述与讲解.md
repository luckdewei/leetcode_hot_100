# 矩阵置零（Set Matrix Zeroes）

- 题目链接：https://leetcode.cn/problems/set-matrix-zeroes/
- 难度：中等
- 标签：数组、哈希表、矩阵

## 题目描述

给定一个 `m x n` 的矩阵，若某元素为 0，则将其所在行和列全部置 0。请使用 **原地** 算法。

### 示例

```
输入：matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出：[[1,0,1],[0,0,0],[1,0,1]]

输入：matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
输出：[[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

### 约束

- `m == matrix.length`, `n == matrix[i].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

---

## 解法一：首行首列作标记（推荐）

### 思路

用第一行、第一列记录该行/列是否该置零。先处理内层 (1..m-1, 1..n-1)，最后根据首行首列标记及原始首行首列是否有零来更新。

### 代码

```python
from typing import List


class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        if not matrix or not matrix[0]:
            return
        m, n = len(matrix), len(matrix[0])
        first_row_zero = any(matrix[0][j] == 0 for j in range(n))
        first_col_zero = any(matrix[i][0] == 0 for i in range(m))
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0
                    matrix[0][j] = 0
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0
        if first_row_zero:
            for j in range(n):
                matrix[0][j] = 0
        if first_col_zero:
            for i in range(m):
                matrix[i][0] = 0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn) |
| 空间 | O(1) |

### 讲解

O(1) 额外空间的关键：首行首列兼做标记数组，用两个变量保存首行首列原始状态。

## 解法二：额外标记数组

### 思路

用 `row_flag` 和 `col_flag` 记录需置零的行列，第二遍统一置零。

### 代码

```python
from typing import List


class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        m, n = len(matrix), len(matrix[0])
        rows = [False] * m
        cols = [False] * n
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    rows[i] = True
                    cols[j] = True
        for i in range(m):
            for j in range(n):
                if rows[i] or cols[j]:
                    matrix[i][j] = 0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn) |
| 空间 | O(m + n) |

### 讲解

思路清晰，但额外 O(m+n) 空间，不满足 follow-up。

## 解法三：直接记录坐标（理解用）

### 思路

收集所有零的坐标，再遍历置零。注意不能直接边遍历边改。

### 代码

```python
from typing import List


class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        zeros = []
        m, n = len(matrix), len(matrix[0])
        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    zeros.append((i, j))
        for i, j in zeros:
            for c in range(n):
                matrix[i][c] = 0
            for r in range(m):
                matrix[r][j] = 0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(mn + k(m+n)) |
| 空间 | O(k) |

### 讲解

边遍历边改会把后续 0 漏标，必须先收集坐标。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 首行首列标记 | O(mn) | O(1) | 面试首选 |
| 标记数组 | O(mn) | O(m+n) | 过渡理解 |
| 坐标集合 | O(mn) | O(k) | 理解陷阱 |

---

## 常见错误

1. **边遍历边置零**：新产生的 0 会错误扩散。
2. **忘记处理首行首列**：应用独立变量记录。
3. **标记与置零顺序错**：先标记内层，最后处理首行首列。
