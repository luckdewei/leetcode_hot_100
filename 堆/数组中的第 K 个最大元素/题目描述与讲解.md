# 数组中的第 K 个最大元素（Kth Largest Element in an Array）

- 题目链接：https://leetcode.cn/problems/kth-largest-element-in-an-array/
- 难度：中等
- 标签：数组、分治、快速选择、堆

## 题目描述

给定整数数组 `nums` 和整数 `k`，请返回数组中第 **k** 个最大的元素。请注意，你需要找的是数组排序后的第 k 个最大元素，而不是第 k 个不同元素。

### 示例

```
输入：nums = [3,2,1,5,6,4], k = 2
输出：5

输入：nums = [3,2,3,1,2,4,5,5,6], k = 4
输出：4
```

### 约束

- `1 <= k <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

---

## 解法一：小根堆维护 K 大（推荐）

### 思路

维护大小为 k 的小根堆，堆顶是 k 个最大元素中最小的，即第 k 大。

### 代码

```python
from typing import List
import heapq


class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        heap: List[int] = []
        for num in nums:
            heapq.heappush(heap, num)
            if len(heap) > k:
                heapq.heappop(heap)
        return heap[0]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n log k) |
| 空间 | O(k) |

### 讲解

比全排序 O(n log n) 更优，k 较小时效率高。面试最常用写法。

## 解法二：QuickSelect

### 思路

快排 partition 思想：随机选 pivot 分区，左边大右边小（或相反），根据 pivot 位置决定递归哪一侧。

### 代码

```python
from typing import List
import random


class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        k = k - 1  # 转为第 k 大对应下标（降序第 k 个）

        def partition(l, r):
            pivot = random.randint(l, r)
            nums[pivot], nums[r] = nums[r], nums[pivot]
            store = l
            for i in range(l, r):
                if nums[i] > nums[r]:
                    nums[store], nums[i] = nums[i], nums[store]
                    store += 1
            nums[store], nums[r] = nums[r], nums[store]
            return store

        def select(l, r):
            pos = partition(l, r)
            if pos == k:
                return nums[pos]
            if pos < k:
                return select(pos + 1, r)
            return select(l, pos - 1)

        return select(0, len(nums) - 1)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | 平均 O(n)，最坏 O(n²) |
| 空间 | O(1) |

### 讲解

平均线性时间，是理论最优。注意随机 pivot 避免最坏情况。

## 解法三：排序

### 思路

降序排序后取第 k-1 个元素。

### 代码

```python
from typing import List


class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        nums.sort(reverse=True)
        return nums[k - 1]
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n log n) |
| 空间 | O(1) 或 O(n) |

### 讲解

最简单，面试可先提再优化为堆或 QuickSelect。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 小根堆 | O(n log k) | O(k) | 面试首选 |
| QuickSelect | 平均 O(n) | O(1) | 追求最优 |
| 排序 | O(n log n) | O(1) | 快速实现 |

## 常见错误

1. **第 k 大 vs 第 k 小**：第 k 大对应降序下标 k-1，或小根堆堆顶。
2. **QuickSelect 边界**：k 从 1 开始，转下标要减 1。
3. **与 Top K 高频混淆**：347 题按频次而非数值大小。
