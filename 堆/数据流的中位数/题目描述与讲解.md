# 数据流的中位数（Find Median from Data Stream）

- 题目链接：https://leetcode.cn/problems/find-median-from-data-stream/
- 难度：困难
- 标签：设计、堆、数据流

## 题目描述

中位数是有序整数列表最中间的那个数。若列表长度偶数，则为中间两数平均值。

实现 `MedianFinder` 类：
- `MedianFinder()` 初始化
- `void addNum(int num)` 将整数加入数据流
- `double findMedian()` 返回目前所有整数的中位数

### 示例

```
输入：["MedianFinder","addNum","addNum","findMedian","addNum","findMedian"]
     [[],[1],[2],[],[3],[]]
输出：[null,null,null,1.50000,null,2.00000]
```

### 约束

- `-10^5 <= num <= 10^5`
- `findMedian` 调用前数据流中至少有一个元素
- 最多 `5 * 10^4` 次调用

---

## 解法一：双堆（推荐）

### 思路

大根堆 `lo` 存较小一半，小根堆 `hi` 存较大一半。保持 `len(lo) >= len(hi)` 且两者长度差不超过 1。中位数由堆顶得出。

### 代码

```python
import heapq


class MedianFinder:
    def __init__(self):
        self.lo = []  # max-heap via negate
        self.hi = []  # min-heap

    def addNum(self, num: int) -> None:
        heapq.heappush(self.lo, -num)
        heapq.heappush(self.hi, -heapq.heappop(self.lo))
        if len(self.hi) > len(self.lo):
            heapq.heappush(self.lo, -heapq.heappop(self.hi))

    def findMedian(self) -> float:
        if len(self.lo) > len(self.hi):
            return float(-self.lo[0])
        return (-self.lo[0] + self.hi[0]) / 2.0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| addNum | O(log n) |
| findMedian | O(1) |
| 空间 | O(n) |

### 讲解

Python 只有小根堆，大根堆用负数模拟。每次先入 `lo` 再平衡，保证 `lo` 堆顶是左半最大值，即中位数左端。

## 解法二：有序列表 + 二分插入

### 思路

维护有序数组，插入时用 `bisect` 找位置，中位数直接取中间元素。

### 代码

```python
import bisect


class MedianFinder:
    def __init__(self):
        self.nums = []

    def addNum(self, num: int) -> None:
        bisect.insort(self.nums, num)

    def findMedian(self) -> float:
        n = len(self.nums)
        mid = n // 2
        if n % 2:
            return float(self.nums[mid])
        return (self.nums[mid - 1] + self.nums[mid]) / 2.0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| addNum | O(n) |
| findMedian | O(1) |
| 空间 | O(n) |

### 讲解

实现简单但插入 O(n)，数据量大时会超时，不适合本题规模。

## 解法三：每次排序（理解用）

### 思路

每次 `addNum` 后重新排序，取中位数。

### 代码

```python
class MedianFinder:
    def __init__(self):
        self.nums = []

    def addNum(self, num: int) -> None:
        self.nums.append(num)
        self.nums.sort()

    def findMedian(self) -> float:
        n = len(self.nums)
        if n % 2:
            return float(self.nums[n // 2])
        return (self.nums[n // 2 - 1] + self.nums[n // 2]) / 2.0
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| addNum | O(n log n) |
| findMedian | O(1) |
| 空间 | O(n) |

### 讲解

仅用于理解，不可通过大规模测试。

|------|--------|------------|----------|
| 双堆 | O(log n) | O(1) | 面试首选 |
| bisect | O(n) | O(1) | 小规模数据 |
| 每次排序 | O(n log n) | O(1) | 理解用 |

## 常见错误

1. **两堆大小关系搞反**：应保证 `lo`（较小半）长度 >= `hi`。
2. **大根堆忘记取负**：Python heapq 是小根堆。
3. **偶数长度中位数**：取两堆顶平均值，注意浮点返回。
