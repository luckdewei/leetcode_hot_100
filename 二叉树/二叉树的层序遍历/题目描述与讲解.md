# 二叉树的层序遍历（Binary Tree Level Order Traversal）

- 题目链接：https://leetcode.cn/problems/binary-tree-level-order-traversal/
- 难度：中等
- 标签：树、广度优先搜索、二叉树

## 题目描述

给你二叉树的根节点 `root`，返回其节点值的**层序遍历**结果（即逐层从左到右访问，每层一个列表）。

### 示例

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]

输入：root = [1]
输出：[[1]]

输入：root = []
输出：[]
```

### 约束

- 树中节点数目在 `[0, 2000]` 范围内
- `-1000 <= Node.val <= 1000`

---

## 解法一：BFS 队列（推荐，同 solution.py）

### 思路

用队列按层扩展：每轮先记录当前层大小 `size`，恰好处理 `size` 个节点即为一层。

### 代码

```python
from collections import deque
from typing import List, Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        res, q = [], deque([root])
        while q:
            level, size = [], len(q)
            for _ in range(size):
                node = q.popleft()
                level.append(node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
            res.append(level)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(w)，w 为最大层宽 |

### 讲解

**`for _ in range(len(q))` 是层序核心**：避免混层。面试中 BFS 模板题，务必熟练。

## 解法二：DFS + 深度分组

### 思路

DFS 遍历时携带深度 `d`，将节点值追加到 `res[d]`。首次到达某深度时扩展结果列表。

### 代码

```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        res = []

        def dfs(node: Optional[TreeNode], depth: int) -> None:
            if not node:
                return
            if depth == len(res):
                res.append([])
            res[depth].append(node.val)
            dfs(node.left, depth + 1)
            dfs(node.right, depth + 1)

        dfs(root, 0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

DFS 也能按层收集，空间为递归栈而非队列。层内顺序为 DFS 序（先左后右），与本题要求一致。

## 解法三：双队列交替（理解 BFS）

### 思路

用两个队列交替存放当前层与下一层节点，当前层处理完毕后交换角色。

### 代码

```python
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        res = []
        cur_level = deque([root])
        while cur_level:
            level, next_level = [], deque()
            while cur_level:
                node = cur_level.popleft()
                level.append(node.val)
                if node.left:
                    next_level.append(node.left)
                if node.right:
                    next_level.append(node.right)
            res.append(level)
            cur_level = next_level
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(w) |

### 讲解

与单队列 + `size` 等价，双队列更直观区分「当前层/下一层」，但代码略长。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| BFS 单队列 | O(n) | O(w) | 面试首选 |
| DFS 分组 | O(n) | O(h) | 已有 DFS 框架时 |
| 双队列 | O(n) | O(w) | 教学演示 |

## 常见错误

1. **未按层分隔**：每弹一个节点就 append 一层，导致每层只有一个元素。
2. **空树返回 `[[]]`**：应返回 `[]`。
3. **层内顺序错误**：应先左后右入队。
