# 二叉树的中序遍历（Binary Tree Inorder Traversal）

- 题目链接：https://leetcode.cn/problems/binary-tree-inorder-traversal/
- 难度：简单
- 标签：栈、树、深度优先搜索、二叉树

## 题目描述

给定一个二叉树的根节点 `root`，返回它的**中序**遍历结果。

中序遍历顺序：**左子树 → 根节点 → 右子树**。

### 示例

```
输入：root = [1,null,2,3]
输出：[1,3,2]

输入：root = []
输出：[]

输入：root = [1]
输出：[1]
```

### 约束

- 树中节点数目在范围 `[0, 100]` 内
- `-100 <= Node.val <= 100`

---

## 解法一：递归 DFS（最易理解）

### 思路

中序遍历的定义本身就是递归：先递归左子树，访问根，再递归右子树。

### 代码

```python
from typing import List, Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node: Optional[TreeNode]) -> None:
            if not node:
                return
            dfs(node.left)
            res.append(node.val)
            dfs(node.right)

        dfs(root)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，每个节点访问一次 |
| 空间 | O(h)，递归栈深度，h 为树高 |

### 讲解

递归写法与遍历定义一一对应，适合初学。面试中可先写递归再优化为迭代。

---

## 解法二：迭代 + 栈（推荐，同 solution.py）

### 思路

用栈模拟递归过程：一路向左压栈；弹出节点时访问，再转向右子树。

### 代码

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res, stack = [], []
        cur = root
        while cur or stack:
            while cur:
                stack.append(cur)
                cur = cur.left
            cur = stack.pop()
            res.append(cur.val)
            cur = cur.right
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

**核心循环不变量**：栈中保存「尚未访问右子树的节点路径」。弹出时即「左子树已处理完毕」的时刻，因此可以安全访问根并进入右子树。

## 解法三：Morris 遍历（O(1) 空间）

### 思路

利用空指针建立临时线索：当前节点无左子树则访问并右移；有左子树则找前驱，若前驱右指针为空则连到当前节点并左移，否则断开并访问当前节点。

### 代码

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []
        cur = root
        while cur:
            if not cur.left:
                res.append(cur.val)
                cur = cur.right
            else:
                pred = cur.left
                while pred.right and pred.right != cur:
                    pred = pred.right
                if not pred.right:
                    pred.right = cur
                    cur = cur.left
                else:
                    pred.right = None
                    res.append(cur.val)
                    cur = cur.right
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(1)，不计输出 |

### 讲解

Morris 遍历适合空间受限场景，但会临时修改树结构（需恢复），面试中了解即可。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 递归 | O(n) | O(h) | 理解定义、快速实现 |
| 迭代栈 | O(n) | O(h) | 面试首选，无栈溢出风险 |
| Morris | O(n) | O(1) | 空间极致优化 |

## 常见错误

1. **前序/中序/后序混淆**：中序是「左-根-右」，前序是「根-左-右」。
2. **迭代时忘记处理右子树**：弹出节点后必须 `cur = cur.right`。
3. **空树返回 `[None]`**：空树应返回 `[]`，不是 `[None]`。
