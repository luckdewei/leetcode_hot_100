# 二叉树的最近公共祖先（Lowest Common Ancestor of a Binary Tree）

- 题目链接：https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/
- 难度：中等
- 标签：树、深度优先搜索、二叉树

## 题目描述

给定一个二叉树，找到两个指定节点 `p` 和 `q` 的**最近公共祖先**（LCA）。

**最近公共祖先**：深度最大的节点，使得 `p` 和 `q` 都是它的后代（节点可以是自身的后代）。

### 示例

```
输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
输出：3

输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 4
输出：5
解释：p 是 q 的祖先

输入：root = [1,2], p = 1, q = 2
输出：1
```

### 约束

- 树中节点数目 `2 <= n <= 10^5`
- `-10^9 <= Node.val <= 10^9`
- 所有 `Node.val` **互不相同**
- `p != q`，且 `p`、`q` 均存在于树中

---

## 解法一：递归后序（推荐，同 solution.py）

### 思路

后序遍历：若当前节点为 p 或 q，返回自身；左右子树分别找 LCA；若两侧均非空，当前即为 LCA；否则返回非空一侧。

### 代码

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def lowestCommonAncestor(
        self, root: TreeNode, p: TreeNode, q: TreeNode
    ) -> TreeNode:
        if not root or root is p or root is q:
            return root
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)
        if left and right:
            return root
        return left or right
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

**精髓**：`left and right` 表示 p、q 分居两侧，则当前根为 LCA。若一侧为空，说明 p、q 在同侧，向上传递非空结果。`root is p` 处理「p 是 q 祖先」的情况。

## 解法二：哈希表记录父节点 + 回溯

### 思路

先 BFS/DFS 记录每个节点的父指针；从 p 向上标记祖先集合；从 q 向上找第一个在集合中的节点。

### 代码

```python
class Solution:
    def lowestCommonAncestor(
        self, root: TreeNode, p: TreeNode, q: TreeNode
    ) -> TreeNode:
        parent = {root: None}
        stack = [root]
        while p not in parent or q not in parent:
            node = stack.pop()
            for child in (node.left, node.right):
                if child and child not in parent:
                    parent[child] = node
                    stack.append(child)
        ancestors = set()
        cur = p
        while cur:
            ancestors.add(cur)
            cur = parent[cur]
        cur = q
        while cur not in ancestors:
            cur = parent[cur]
        return cur
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

需两次遍历 + 额外空间，但思路直观，适合无法递归的环境。

## 解法三：BST 特化（236 为普通二叉树）

### 思路

若改为 BST：从根出发，若 p、q 均小于根则进左；均大于根则进右；否则当前为 LCA。

### 代码

```python
class Solution:
    def lowestCommonAncestor(
        self, root: TreeNode, p: TreeNode, q: TreeNode
    ) -> TreeNode:
        while root:
            if p.val < root.val and q.val < root.val:
                root = root.left
            elif p.val > root.val and q.val > root.val:
                root = root.right
            else:
                return root
        return root
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(h) |
| 空间 | O(1) |

### 讲解

236 题为普通二叉树，不能用 BST 性质；235 题可直接用此法。对比两题加深理解。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 递归后序 | O(n) | O(h) | 236 面试首选 |
| 父指针哈希 | O(n) | O(n) | 需多次查询 |
| BST 特化 | O(h) | O(1) | 235 题 |

---

## 常见错误

1. **比较 val 而非节点引用**：236 题应用 `root is p`，值可能重复（235 除外）。
2. **p、q 同侧却返回 null**：应 `return left or right`。
3. **混淆 235 与 236**：BST 可走 O(h) 路径。
