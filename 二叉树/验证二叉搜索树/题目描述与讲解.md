# 验证二叉搜索树（Validate Binary Search Tree）

- 题目链接：https://leetcode.cn/problems/validate-binary-search-tree/
- 难度：中等
- 标签：树、深度优先搜索、二叉搜索树、二叉树

## 题目描述

给你一个二叉树的根节点 `root`，判断其是否是一个**有效的二叉搜索树**（BST）。

BST 定义：左子树所有节点 < 根 < 右子树所有节点，且左右子树也必须是 BST。

### 示例

```
输入：root = [2,1,3]
输出：true

输入：root = [5,1,4,null,null,3,6]
输出：false
解释：根 5，右子树中 4 < 5，违反 BST
```

### 约束

- 树中节点数目范围 `[1, 10^4]`
- `-2^31 <= Node.val <= 2^31 - 1`

---

## 解法一：中序遍历递增（推荐，同 solution.py）

### 思路

BST 的中序遍历严格递增。DFS 中序，用 `prev` 记录上一个值，若 `node.val <= prev` 则非法。

### 代码

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        self.prev = float("-inf")

        def dfs(node: Optional[TreeNode]) -> bool:
            if not node:
                return True
            if not dfs(node.left):
                return False
            if node.val <= self.prev:
                return False
            self.prev = node.val
            return dfs(node.right)

        return dfs(root)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

**必须用严格递增**（`<` 而非 `<=`），否则 duplicate 会误判。`prev` 初值用 `-inf` 兼容负值根。

## 解法二：递归传递上下界

### 思路

每个节点必须在 `(min_val, max_val)` 开区间内。左子树上界为 `root.val`，右子树下界为 `root.val`。

### 代码

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        def valid(node: Optional[TreeNode], lo: float, hi: float) -> bool:
            if not node:
                return True
            if not (lo < node.val < hi):
                return False
            return valid(node.left, lo, node.val) and valid(
                node.right, node.val, hi
            )

        return valid(root, float("-inf"), float("inf"))
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

此法更直接体现 BST 定义，避免「只比较父节点」的经典陷阱（如 5-1-4 反例）。

## 解法三：迭代中序

### 思路

用栈实现中序，同样检查相邻值严格递增。

### 代码

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        stack, prev, cur = [], float("-inf"), root
        while cur or stack:
            while cur:
                stack.append(cur)
                cur = cur.left
            cur = stack.pop()
            if cur.val <= prev:
                return False
            prev = cur.val
            cur = cur.right
        return True
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

与递归中序等价，适合深树或需迭代风格时。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 中序递增 | O(n) | O(h) | 简洁，同 solution |
| 上下界 | O(n) | O(h) | 体现 BST 定义 |
| 迭代中序 | O(n) | O(h) | 避免递归 |

## 常见错误

1. **只比较节点与直接父节点**：子树其他节点可能越界。
2. **使用 `<=` 允许相等**：BST 通常要求严格小于/大于。
3. **`prev` 初值为 0**：负值根会误判，应用 `-inf`。
