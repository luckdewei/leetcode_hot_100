# 二叉树的右视图（Binary Tree Right Side View）

- 题目链接：https://leetcode.cn/problems/binary-tree-right-side-view/
- 难度：中等
- 标签：树、深度优先搜索、广度优先搜索、二叉树

## 题目描述

给定一棵二叉树的根节点 `root`，想象自己站在树的右侧，返回从右侧能看到的节点值（从上到下）。

### 示例

```
输入：root = [1,2,3,null,5,null,4]
输出：[1,3,4]

输入：root = [1,null,3]
输出：[1,3]

输入：root = []
输出：[]
```

### 约束

- 二叉树的节点个数的范围 `[0, 100]`
- `-100 <= Node.val <= 100`

---

## 解法一：BFS 层序取最后（推荐，同 solution.py）

### 思路

层序遍历每一层，该层最后一个出队节点即为右视图可见节点。

### 代码

```python
from collections import deque
from typing import List, Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        res, q = [], deque([root])
        while q:
            size = len(q)
            for i in range(size):
                node = q.popleft()
                if i == size - 1:
                    res.append(node.val)
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(w) |

### 讲解

右视图 = 每层最右节点。注意左子树更深时，右视图可能是左子树的节点（见示例 1 中 5 不可见而 4 可见）。

## 解法二：DFS 先右后左

### 思路

按「根-右-左」DFS，首次到达某深度时记录该节点（该深度最先访问的是最右）。

### 代码

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node: Optional[TreeNode], depth: int) -> None:
            if not node:
                return
            if depth == len(res):
                res.append(node.val)
            dfs(node.right, depth + 1)
            dfs(node.left, depth + 1)

        dfs(root, 0)
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

`depth == len(res)` 保证每层只记录第一次（最右）访问。先右后左是关键。

## 解法三：BFS 右子优先入队

### 思路

每层仍 BFS，但入队顺序改为先右后左，则每层第一个出队即为右视图。

### 代码

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        res, q = [], deque([root])
        while q:
            level_size = len(q)
            for i in range(level_size):
                node = q.popleft()
                if i == 0:
                    res.append(node.val)
                if node.right:
                    q.append(node.right)
                if node.left:
                    q.append(node.left)
            q = deque(list(q)[level_size:])  # 简化：标准写法见解法一
        return res
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(w) |

### 讲解

更简洁的变体：每层循环中第一个处理的节点即答案，配合先右后左入队。生产代码仍推荐解法一的 `i == size-1`。

---

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| BFS 取最后 | O(n) | O(w) | 直观，同 solution |
| DFS 先右 | O(n) | O(h) | 代码短 |
| BFS 先右 | O(n) | O(w) | 与 DFS 对称 |

---

## 常见错误

1. **只取根到叶最右路径**：右视图不等于右链，左子树深层节点可能可见。
2. **DFS 先左后右**：会记录左视图。
3. **空层处理**：空树返回 `[]`。
