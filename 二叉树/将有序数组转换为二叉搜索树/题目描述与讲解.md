# 将有序数组转换为二叉搜索树（Convert Sorted Array to Binary Search Tree）

- 题目链接：https://leetcode.cn/problems/convert-sorted-array-to-binary-search-tree/
- 难度：简单
- 标签：树、二叉搜索树、数组、分治、二叉树

## 题目描述

给你一个整数数组 `nums`（**升序无重复**），将其转换为**高度平衡**的二叉搜索树。

高度平衡：每个节点左右两个子树的高度差不超过 1。

### 示例

```
输入：nums = [-10,-3,0,5,9]
输出：[0,-3,9,-10,null,5]
解释：[0,-10,5,null,-3,null,9] 也合法

输入：nums = [1,3]
输出：[3,1] 或 [1,null,3]
```

### 约束

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 按升序排列且无重复

---

## 解法一：分治递归（推荐，同 solution.py）

### 思路

BST 中序为升序。选中间元素为根，左侧建左子树，右侧建右子树，保证平衡。

### 代码

```python
from typing import List, Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        def build(lo: int, hi: int) -> Optional[TreeNode]:
            if lo > hi:
                return None
            mid = (lo + hi) // 2
            node = TreeNode(nums[mid])
            node.left = build(lo, mid - 1)
            node.right = build(mid + 1, hi)
            return node

        return build(0, len(nums) - 1)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，每个元素建一个节点 |
| 空间 | O(log n) 递归栈；树高 O(log n) |

### 讲解

选中间值使左右子树规模尽量相等，高度差 ≤ 1。这是「分治 + BST 性质」的经典结合。

## 解法二：中序模拟 + 前序建树

### 思路

先确定 BST 前序遍历顺序：每次选区间中点作为根，再递归左右。与解法一等价，可显式用前序数组辅助理解。

### 代码

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        preorder = []

        def build(lo: int, hi: int) -> Optional[TreeNode]:
            if lo > hi:
                return None
            mid = (lo + hi) // 2
            preorder.append(nums[mid])
            root = TreeNode(nums[mid])
            root.left = build(lo, mid - 1)
            root.right = build(mid + 1, hi)
            return root

        root = build(0, len(nums) - 1)
        return root
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

帮助理解「前序根-左-右」与「中序升序」的对应关系，与 105 题构造思想一脉相承。

## 解法三：迭代 + 栈（了解）

### 思路

用栈保存 `(lo, hi, parent, is_left)`，模拟递归建树过程。

### 代码

```python
class Solution:
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        root = TreeNode(nums[(len(nums) - 1) // 2])
        stack = [(0, len(nums) - 1, root)]
        while stack:
            lo, hi, node = stack.pop()
            mid = (lo + hi) // 2
            node.val = nums[mid]
            if mid + 1 <= hi:
                node.right = TreeNode()
                stack.append((mid + 1, hi, node.right))
            if lo <= mid - 1:
                node.left = TreeNode()
                stack.append((lo, mid - 1, node.left))
        return root
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

迭代版较少考，掌握递归分治即可。

## 常见错误

1. **选左端或右端为根**：会得到链状树，不平衡。
2. **区间边界 off-by-one**：左子树 `[lo, mid-1]`，右子树 `[mid+1, hi]`。
3. **允许重复值**：本题无重复，若有重复需明确 BST 规则。
