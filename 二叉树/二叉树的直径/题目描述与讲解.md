# 二叉树的直径（Diameter of Binary Tree）

- 题目链接：https://leetcode.cn/problems/diameter-of-binary-tree/
- 难度：简单
- 标签：树、深度优先搜索、二叉树

## 题目描述

给定一棵二叉树，计算其**直径长度**。直径是树中任意两个节点之间最长路径的**边数**。

注意：路径可能不经过根节点。

### 示例

```
输入：root = [1,2,3,4,5]
输出：3
解释：最长路径 4 ↔ 2 ↔ 5，共 3 条边

输入：root = [1,2]
输出：1
```

### 约束

- 树中节点数目在范围 `[1, 10^4]` 内
- `-100 <= Node.val <= 100`

---

## 解法一：后序 DFS 一次遍历（推荐，同 solution.py）

### 思路

对每个节点，经过该节点的最长路径 = 左子树深度 + 右子树深度（边数）。DFS 返回子树深度，同时用全局变量更新直径。

### 代码

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        self.ans = 0

        def depth(node: Optional[TreeNode]) -> int:
            if not node:
                return 0
            left = depth(node.left)
            right = depth(node.right)
            self.ans = max(self.ans, left + right)
            return 1 + max(left, right)

        depth(root)
        return self.ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

**关键**：`depth` 返回的是节点数深度，而 `left + right` 恰好是边数直径。后序保证计算父节点时左右深度已知。

---

## 解法二：两遍 DFS（理解用）

### 思路

第一遍求每个节点深度（可缓存）；第二遍对每个节点计算 `depth(left) + depth(right)` 取最大。

### 代码

```python
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        depth_cache = {}

        def depth(node: Optional[TreeNode]) -> int:
            if not node:
                return 0
            if node in depth_cache:
                return depth_cache[node]
            d = 1 + max(depth(node.left), depth(node.right))
            depth_cache[node] = d
            return d

        ans = 0

        def walk(node: Optional[TreeNode]) -> None:
            nonlocal ans
            if not node:
                return
            ans = max(ans, depth(node.left) + depth(node.right))
            walk(node.left)
            walk(node.right)

        walk(root)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

思路直观但多一次遍历；解法一在单次后序中同时完成，更优。

## 解法三：BFS 求最远节点（了解）

### 思路

从任意节点 BFS 找最远节点 u，再从 u BFS 找最远节点 v，u-v 距离即为直径（无负权边时成立）。需两次 BFS + 重建路径，实现较繁。

### 代码

```python
from collections import deque


class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        def bfs_farthest(start: Optional[TreeNode]):
            parent = {start: None}
            q = deque([start])
            last = start
            while q:
                node = q.popleft()
                last = node
                for child in (node.left, node.right):
                    if child and child not in parent:
                        parent[child] = node
                        q.append(child)
            dist = 0
            cur = last
            while parent[cur]:
                dist += 1
                cur = parent[cur]
            return last, dist

        _, _ = bfs_farthest(root)
        _, diameter = bfs_farthest(root)
        return diameter
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n) |

### 讲解

两次 BFS 是图论求树直径的经典方法，二叉树场景下不如 DFS 简洁。

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 后序 DFS | O(n) | O(h) | 面试首选 |
| 两遍 DFS | O(n) | O(n) | 理解分解 |
| 两次 BFS | O(n) | O(n) | 图论背景 |

## 常见错误

1. **把直径当成节点数**：题目要的是边数，深度差即边数。
2. **只考虑经过根的路径**：直径可能在子树内。
3. **深度定义不一致**：返回深度时用节点数，更新直径时用边数 `left + right`。
