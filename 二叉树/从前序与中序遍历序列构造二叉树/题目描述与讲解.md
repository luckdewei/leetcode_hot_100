# 从前序与中序遍历序列构造二叉树（Construct Binary Tree from Preorder and Inorder Traversal）

- 题目链接：https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/
- 难度：中等
- 标签：树、数组、哈希表、分治、二叉树

## 题目描述

给定两个整数数组 `preorder` 和 `inorder`，其中 `preorder` 是二叉树的**前序**遍历，`inorder` 是同一棵树的**中序**遍历，构造并返回二叉树。

### 示例

```
输入：preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出：[3,9,20,null,null,15,7]

输入：preorder = [-1], inorder = [-1]
输出：[-1]
```

### 约束

- `1 <= preorder.length <= 3000`
- `inorder.length == preorder.length`
- `-3000 <= preorder[i], inorder[i] <= 3000`
- `preorder` 和 `inorder` 均无重复

---

## 解法一：递归 + 哈希定位根（推荐，同 solution.py）

### 思路

前序首元素为根；在中序中找到根的位置 `mid`，左侧为左子树（长度 `left_size`），右侧为右子树。递归划分区间建树。

### 代码

```python
from typing import List, Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        idx = {v: i for i, v in enumerate(inorder)}

        def build(pre_lo: int, pre_hi: int, in_lo: int, in_hi: int) -> Optional[TreeNode]:
            if pre_lo > pre_hi:
                return None
            root_val = preorder[pre_lo]
            mid = idx[root_val]
            left_size = mid - in_lo
            root = TreeNode(root_val)
            root.left = build(pre_lo + 1, pre_lo + left_size, in_lo, mid - 1)
            root.right = build(pre_lo + left_size + 1, pre_hi, mid + 1, in_hi)
            return root

        return build(0, len(preorder) - 1, 0, len(inorder) - 1)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，哈希 O(1) 定位 |
| 空间 | O(n)，哈希 + 递归栈 O(h) |

### 讲解

**区间索引是核心**：前序左子树 `[pre_lo+1, pre_lo+left_size]`，右子树 `[pre_lo+left_size+1, pre_hi]`。哈希表避免每次 O(n) 找根。

## 解法二：递归 + 中序线性扫描

### 思路

不用哈希，每次在前序取根，在中序中线性查找 `mid`。逻辑相同，适合理解。

### 代码

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        if not preorder:
            return None
        root_val = preorder[0]
        mid = inorder.index(root_val)
        root = TreeNode(root_val)
        root.left = self.buildTree(preorder[1 : 1 + mid], inorder[:mid])
        root.right = self.buildTree(preorder[1 + mid :], inorder[mid + 1 :])
        return root
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²)，每次 index 扫描 |
| 空间 | O(n) |

### 讲解

切片写法简洁但慢；面试应用解法一的索引版。

## 解法三：迭代 + 栈（了解）

### 思路

用栈模拟递归：前序首为根入栈；中序指针递增，当前根在中序位置左侧的连续节点作为右子树链压栈。

### 代码

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        if not preorder:
            return None
        root = TreeNode(preorder[0])
        stack = [root]
        in_idx = 0
        for i in range(1, len(preorder)):
            node = stack[-1]
            if node.val != inorder[in_idx]:
                node.left = TreeNode(preorder[i])
                stack.append(node.left)
            else:
                while stack and stack[-1].val == inorder[in_idx]:
                    node = stack.pop()
                    in_idx += 1
                node.right = TreeNode(preorder[i])
                stack.append(node.right)
        return root
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |

### 讲解

迭代版较难手写，掌握递归 + 哈希即可应对绝大多数面试。

## 常见错误

1. **左右子树区间算错**：`left_size = mid - in_lo`，不是 `mid`。
2. **前序根位置错误**：左子树前序从 `pre_lo+1` 开始。
3. **有重复值**：本题无重复；若有重复需额外信息。
