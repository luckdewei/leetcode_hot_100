# 路径总和 III（Path Sum III）

- 题目链接：https://leetcode.cn/problems/path-sum-iii/
- 难度：中等
- 标签：树、深度优先搜索、哈希表、二叉树

## 题目描述

给定一个二叉树的根节点 `root` 和一个整数 `targetSum`，求该树里**不需要从根到叶**的路径数目，使得路径上节点值之和等于 `targetSum`。

路径从任意节点出发，向下（只能到子节点）结束于任意节点。

### 示例

```
输入：root = [10,5,-3,3,2,null,11,3,-2,null,1], targetSum = 8
输出：3
解释：路径 5→3、5→2→1、-3→11

输入：root = [1], targetSum = 1
输出：1
```

### 约束

- 二叉树的节点个数的范围 `[0, 1000]`
- `-10^9 <= Node.val <= 10^9`
- `-1000 <= targetSum <= 1000`

---

## 解法一：前缀和 + 哈希表（推荐，同 solution.py）

### 思路

从根到当前节点的路径和为 `cur`。若存在祖先前缀和 `prev` 满足 `cur - prev = targetSum`，则存在一条以当前节点结尾的路径。用哈希表记录各前缀和出现次数，回溯时撤销。

### 代码

```python
from collections import defaultdict
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        self.ans = 0
        prefix = defaultdict(int)
        prefix[0] = 1

        def dfs(node: Optional[TreeNode], cur: int) -> None:
            if not node:
                return
            cur += node.val
            self.ans += prefix[cur - targetSum]
            prefix[cur] += 1
            dfs(node.left, cur)
            dfs(node.right, cur)
            prefix[cur] -= 1

        dfs(root, 0)
        return self.ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(n)，哈希表 + 递归栈 |

### 讲解

与「和为 K 的子数组」（560）同框架。`prefix[0]=1` 处理从根开始的路径。回溯必须 `prefix[cur]-=1`。

## 解法二：双重 DFS（暴力）

### 思路

以每个节点为起点 DFS，累加向下路径和，等于 target 则计数。

### 代码

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        def count_from(node: Optional[TreeNode], remain: int) -> int:
            if not node:
                return 0
            cnt = 1 if node.val == remain else 0
            remain -= node.val
            return cnt + count_from(node.left, remain) + count_from(node.right, remain)

        def dfs(node: Optional[TreeNode]) -> int:
            if not node:
                return 0
            return count_from(node, targetSum) + dfs(node.left) + dfs(node.right)

        return dfs(root)
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²)，最坏链状树 |
| 空间 | O(h) |

### 讲解

直观易写，小数据可通过，面试应升级到前缀和。

## 解法三：前缀和 + 数组枚举（理解用）

### 思路

DFS 维护从根到当前的路径值列表，对每个节点枚举以之为结尾的所有连续子段求和（等价暴力，便于理解）。

### 代码

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        self.ans = 0
        path = []

        def dfs(node: Optional[TreeNode]) -> None:
            if not node:
                return
            path.append(node.val)
            s = 0
            for i in range(len(path) - 1, -1, -1):
                s += path[i]
                if s == targetSum:
                    self.ans += 1
            dfs(node.left)
            dfs(node.right)
            path.pop()

        dfs(root)
        return self.ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n²) |
| 空间 | O(h) |

### 讲解

每个节点向后枚举子段，帮助理解「不要求从根出发」的含义。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 前缀和哈希 | O(n) | O(n) | 面试首选 |
| 双重 DFS | O(n²) | O(h) | 暴力理解 |
| 路径枚举 | O(n²) | O(h) | 直观子段 |

## 常见错误

1. **只统计根到叶路径**：本题可从任意节点开始、任意节点结束。
2. **忘记回溯**：`prefix[cur]` 必须在离开节点时减 1。
3. **未初始化 prefix[0]**：漏计从根开始的路径。
