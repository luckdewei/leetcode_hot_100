# 无重复字符的最长子串（Longest Substring Without Repeating Characters）

- 题目链接：https://leetcode.cn/problems/longest-substring-without-repeating-characters/
- 难度：中等
- 标签：哈希表、字符串、滑动窗口

## 题目描述

给定一个字符串 `s`，请你找出其中不含有重复字符的**最长子串**的长度。

### 示例

```
输入：s = "abcabcbb"
输出：3
解释：最长子串是 "abc"，长度为 3。

输入：s = "bbbbb"
输出：1
解释：最长子串是 "b"，长度为 1。

输入：s = "pwwkew"
输出：3
解释：最长子串是 "wke"，长度为 3。
     请注意，你的答案必须是子串的长度，"pwke" 是一个子序列，不是子串。
```

### 约束

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成

---

## 解法一：暴力枚举（理解用）

### 思路

枚举所有子串 `[i, j]`，检查是否有重复字符。

### 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        n = len(s)
        ans = 0
        for i in range(n):
            seen = set()
            for j in range(i, n):
                if s[j] in seen:
                    break
                seen.add(s[j])
                ans = max(ans, j - i + 1)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n³) 或 O(n²)（配合 set 检查） |
| 空间 | O(min(n, |Σ|)) |

### 讲解

n 最大 5×10⁴，暴力会超时，仅作理解。

---

## 解法二：滑动窗口 + 哈希表（推荐，最优）

### 思路

维护窗口 `[left, right]`，保证内部无重复字符。

- `right` 右扩，把 `s[right]` 加入窗口
- 若 `s[right]` 重复且上次出现位置 `≥ left`，则 `left` 跳到 `上次位置 + 1`
- 用哈希表 `last` 记录每个字符最近出现的下标
- 每次更新最大长度 `right - left + 1`

### 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last = {}
        left = 0
        ans = 0

        for right, ch in enumerate(s):
            if ch in last and last[ch] >= left:
                left = last[ch] + 1
            last[ch] = right
            ans = max(ans, right - left + 1)

        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，right 遍历一次，left 最多移动 n 次 |
| 空间 | O(min(n, |Σ|))，字符集大小 |

### 讲解

**示例（s = "pwwkew"）：**

```
right=0 'p': 窗口 "p", len=1
right=1 'w': 窗口 "pw", len=2
right=2 'w': 重复，left 从 0 跳到 2，窗口 "w", len=1
right=3 'k': 窗口 "wk", len=2
right=4 'e': 窗口 "wke", len=3  ← 答案
right=5 'w': 重复，left 跳到 3，窗口 "kew", len=3
```

**为什么用「上次下标」而不是 set？**  
当 left 右移后，更早的重复字符已在窗口外，set 需要手动删除；记录下标可 O(1) 判断重复是否落在当前窗口内。

**窗口内无重复的不变式：**  
任意时刻 `[left, right]` 中每个字符最多出现一次。

## 解法三：滑动窗口 + 集合

### 思路

用 `set` 维护窗口字符。`right` 扩展时若字符已存在，则 `left` 不断右移并从中删除字符，直到无重复。

### 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        window = set()
        left = 0
        ans = 0
        for right, ch in enumerate(s):
            while ch in window:
                window.remove(s[left])
                left += 1
            window.add(ch)
            ans = max(ans, right - left + 1)
        return ans
```

### 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n)，每个字符最多入集、出集各一次 |
| 空间 | O(min(n, |Σ|)) |

### 讲解

与哈希表法等价，逻辑更直观。面试两种都可讲。

## 解法对比

| 解法 | 时间 | 空间 | 适用场景 |
|------|------|------|----------|
| 暴力 | O(n²) | O(n) | 理解题意 |
| 哈希表 + 窗口 | O(n) | O(字符集) | 代码简洁，首选 |
| 集合 + 窗口 | O(n) | O(字符集) | 直观易懂 |

## 常见错误

1. **子串与子序列混淆**：必须是连续子串，不能跳着选字符。
2. **重复时 left 只 +1**：应跳到「重复字符上次位置 + 1」，否则窗口仍含重复。
3. **空字符串**：应返回 0。
4. **未判断重复位置是否在窗口内**：若上次出现位置 < left，不算重复。
